My Stop Doing List

发表于 2023-02-20

一、投资交易

  1. 理性交易，永远别赌！欲速则不达！

  2. 不懂不做！认清自己的能力边界，只做自己能力圈内的交易，就足够发财了！

  3. 不要想着一夜暴富，渴望一夜暴富的想法非常危险，因为这意味着铤而走险！

二、两性关系

  1. 不要睡免费的女人！免费的东西往往是最贵的，因为有很高的隐性成本！

  2. 找各种理由向你索要大额财产的女人，多半居心不良，要马上让她滚！你有一个包子，狗会跟着你，你把包子给了狗，狗就会离你而去！

  3. 女人永远慕强，永远不要忘记这一点！不要被女人PUA了！

  4. 女人最擅长的，就是利用感情和性获取利益！这是女性的本能！

三、为人处世

  1. 与人交流时，不要用反问句！反问句往往意味着轻视对方，是交流的大忌。

  2. 不要逞口舌之快，更不要与陌生人发生正面冲突，万一对方是个极端分子呢？

  3. 不要装逼，更不要炫富。炫富装逼时，获得的仅是一点虚荣心，但却可能引起贼惦记！

  4. 不贪小便宜，贪小便宜吃大亏。太便宜的东西或服务，里面往往有超级大坑！只要不贪小便宜，就能自动规避这世界上很多坑和骗局！

  5. 出门在外，不要帮助陌生人，有困难可以找警察！

四、健康养生

  1. 早起早睡，不懒床！

  2. 手机不上床，色情严提防。

  3. 不要纵欲，同时改掉手淫的坏习惯！最好的方法就是防微杜渐，只要不开始第一步，就永远不会走到第二步！

五、交通出行

  1. 出门在外，安全第一。失去生命失去一切！

  2. 不要闯红灯，人生路漫漫，耽搁几分钟没什么关系。

  3. 走路不要听歌或玩手机，马路上行人是最弱势的群体，要好好保护自己。

  4. 打车时不要图便宜拼车，浪费时间，影响心情，同时也失去了打车的意义！

六、识人善任

  1. 巧言令色，鲜亦仁！

  2. 反复小人，不可用也！
