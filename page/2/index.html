科学上网的两种方法

发表于 2023-02-25

一、远程主机

  1. 通过阿里云或腾讯云买一台墙外的Windows主机，然后登录上主机即可直接翻墙。也可以购买一台Ubuntu主机，然后自行安装桌面软件。

  2. 这种方法的优点操作简单，不需要任何配置，也不用担心被封。缺点是使用不方便，低配主机往往分辨率较差。

二、SSH隧道

  1. 购买一台外网服务器：比如可以在阿里云或腾讯云上购买一台新加坡地区的轻量应用服务器，每月仅需30元左右。

  2. 配置SSH免密登录：通过ssh keygen命令生成本地密钥，并将公钥id_rsa.pub中的内容复制到服务器下的.ssh/authorized_keys文件内即可。

  3. 开启SSH隧道：命令ssh -D port username@server_ip，比如ssh -D 2275 lighthouse@129.229.199.197，表示建立一个端口为2275的隧道。

  4. 本地监听上述端口：Mac电脑操作路径为：System Preferences > Network > Advanced > Proxies，勾选SOCKS Proxies并在右侧SOCKS Proxy Server中填入127.0.0.1和2275。然后点击OK返回，再点击Network的Apply即可。

  5. Telegram代理配置：Settings > Data and Storage > Use Proxy > Add Proxy，选择SOCKS5，SERVER输入127.0.0.1，PORT输入2275即可。

  6. 关闭SSH隧道：强制关闭本地命令行窗口即可。

市场走势可以被精确预测吗？

发表于 2023-02-16

  1. 在2023年以前，我一直坚定的认为市场走势是可以被精确预测的。因为2008年我炒股票时，在鼎砥论坛见识了百年一人的全息未来线，对股票市场做出过长期精准的预测，惊为天人！

  2. 截止2022年，百年一人已经研究市场预测24年了，功力已经非常深厚，也从市场上赚到了大量的财富。但看其最近2年的预测，却仍然不尽完美，无法做到100%的确定性。比如2022.02.28预测美元指数已经外强中干了，但事实上那时正是美元指数大涨的起点，一直又涨了7个月才见顶。

  3. 最近沉迷斗地主，我试图在其中找到一种可以稳定获胜的模式，花了很多时间思考和研究，最终也没有成功。因为在一个博弈系统中，参与者的行为会直接影响对手行为，从而再次作用到自身之上，即一但你按预测行事，整个系统就就会发生变化，从而使你的预测失效。

  4. 这让我想到交易市场也是如此，市场走势是多空博弈的结果，而多空博弈又需要保持平衡。一旦大家在某个预期的点位达成一致，那这个位置的多空力量必定失衡，从而导致市场根本无法运行到这个点位。

  5. 由于预测市场的行为会对反过来作用于自身，且这种相互作用的关系会层层嵌套，这导致技术分析永远无法精确地预测市场的走势，只能大致的知晓市场未来的运行状态。理论上来说，只要是由自主个体参与的博弈市场，都是不可精确预测的。

  6. 但这并不能完全否定技术分析的作用，虽然不能精确预测，但模糊预测还是可以的。比如用于判断当前趋势已经运行到了哪个阶段。所以，技术分析仍是多空博弈中非常有效的工具。

段永平投资理念(摘录)

发表于 2023-02-09

一. 不懂不做

  1. 我问过巴菲特在投资中不可以做的事情是什么，他告诉我说：“不做空，不借钱，最重要的是不要做不懂的东西。”

  2. 做我们懂的事，失败的概率就小；做我们不懂的事，失败的概率就大；即使我们很懂的事，仍然有失败的概率。

  3. 我不懂的东西我不碰，然后很多人就说了，这个机会可能会失去，等你弄懂了，它可能已经涨上去了。我说，又如何？一辈子你有非常多的机会，但是你要抓快，你可能就掉进去了，这就是快是慢的一个概念。

  4. 这些年，我在投资里亏掉的美金数以亿计，每一笔都是违背老巴教导的情况下亏的，而赚到的大钱也都是在自己真正懂的地方赚的。作为刚出道的学生，书上的东西可能知道的很多，但融到骨子里还需要吃很多亏后才行。所以，如果你马上投入投资行业，最重要的是要保守啊，别因为一个错误就再也爬不起来了。

  5. 投资最重要的是投在你真正懂的东西上。无论学历高低，一个人总会懂些什么，而你懂的东西可能有一天会让你发现机会。我自己抓住的机会也好像和学历没什么必然的联系。

  6. 知道自己能力圈有多大或者说知道自己能力圈的边界在哪里要远远重要过自己能力圈有多大，这也是人们经常能看到很多很“聪明人”的投资表现长期不好的原因。当然，这些“聪明人”会把别人的成功或自己的不成功归结于运气或“accidents”，而且他们总是能够很“聪明”地找到办法让自己认为确实如此。功夫熊猫里说”There are no accidents”其实是蛮有道理的。

二. 慢即是快

  1. 我记得以前有个人问过我“危机时刻你怎么办？”，我就跟他举了个例子：我开个车时速200公里，前面有堵墙，我马上就要撞上了，你说我能怎么办？没办法！肯定就死定了！最重要的是你不要把车开那么快。

三. 有所为有所不为

  1. 巴菲特之所以能够如此厉害，最重要的一点就是他能坚持不做他认为不对的事情。要坚持这点其实非常不容易，因为往往“不对的事情”往往是有短期诱惑的。

  2. 我觉得投资最重要的是你不要亏钱，而不是说你要去抓住一个这样的机会或者那样的机会，机会永远都在，就是你抓住机会的能力是有限的，每个人抓住机会的能力不一样，所以最重要是你要知道自己什么事情能做，什么事情不能做，尤其是不能做这个东西特别重要。你可能99件事情做对了，一件错误就能够把你99件的这个业绩全部给葬送掉。

四. 更健康更长久

  1. 我们企业最核心的价值观：更健康，更长久。

  2. 投资和别人怎么看是没有任何关系的，只要你能看的稍微长远一点就行了。

  3. Right kind of business, right people, right price.只要有了这三点，长期而言投资结果必然不错。

有哪些无风险的赚钱机会？

发表于 2023-02-05

一. ETH质押挖矿

  1. 目前，ETH质押挖矿的年化收益约4%~5%，虽然不高，但胜在稳定，而且是币本位，相当于额外赚币了！

  2. 同时BETH支持质押借币，且质押借币期间，仍可享受质押挖矿收益。利用这一点，可以通过循环质押提高收益。操作流程如下：将ETH质押得到的BETH，将BETH质押借出ETH，借出的ETH又可以再次质押得到BETH…如此循环操作。

  3. 比如初始资金为100ETH，质押挖矿收益为4.5%，质押借币的利率为2.5%，能借出60%的币。则大体质押循环流程如下：100ETH ➝ 100BETH ➝ 60ETH ➝ 60BETH ➝ 36ETH ➝ 36BETH ➝ 20ETH ➝ 20BETH。这样几轮操作下来后，共质押了216BETH，其中有106ETH需要支持利息。则净质押收益为：(216*4.5% - 106*2.5%)/100 = 7.07%。而且因为BETH和ETH的强挂钩，质押借币订单几乎没有强平风险。

  4. 另外，等ETH质押可以提币后，还可以进行价差套利。即低价买入BETH，再通过质押提币将BETH1:1兑换为ETH。

二. 现货网格交易

  1. 现货网格：现货网格的缺点是资金利用效率低，优点是只要有波动就会有收益，而且可以将收益提取出来补充现金流。

  2. 网格下限：现货网格最重要的是网格下限的设置，要保证网格下限不被击穿，长期来看是稳赚不赔的，而且收益非常不错的。

  3. 网格间距：网格交易一买一卖，大约需要0.2%的手续费，所以，建议每格利润保持在1%~2%以上，可以最有效地捕捉价格波动。

  4. 网格类型：现货网格分为等差和等比两种类型，建议两种网格各开一个，在实践在观察二者的表现，后续再确实。我目前倾向于等比类型。

三. 双币赢网格策略

  1. 双币赢是一个非常好的产品，在一定情况下，可以实现确定的无风险收益。比如当ETH价格为1000U，我很确定行情已见底，接下来要么震荡，要么上涨，且至少会涨到2000U。那么在ETH涨到2000U之前，我就可以尽情地操作双币赢收取利息。

  2. 若是不想分析行情，还可以将标的价格划分为不同的安全等级，根据安全等级的高低决定投入资金的多少。详情参考《基于双币赢的投资策略》。

  3. 最简单的策略就是以历史最高价的1/2为分界线，中位线以上为高风险区域，中位线以下为低风险区域。只做目标价在中位线以下的双币赢产品，即可实现牛市赚U，熊市赚币。长期来看，一定能实现稳赚不赔。

四. 马丁格尔网格策略

  1. 利用币价“下跌空间有限，上涨空间无限”的不对称性，结合加倍开仓法，可以实现100%的交易胜率。参考《马丁格尔网格交易策略》。

  2. 这种策略的关键是对网格数量的划分，可以根据当前币价的位置决定网格数量。比如规定网格最大数量为n，则公式为网格数量=当前价*n/历史最高价。比如ETH的历史最高价为4800，当前价为1600，最大网格数量为4，则当前网格数量应为1600*4/4800=1.33，然后根据实际情况选择向上取整还是向下取整即可。

五. 期现价差套利

  1. 在牛市中，次季合约与现货的差价往往会高达10%以上，此时入场做多现货，卖空次季合约，可以赚取确定的差价。在熊市中，次季合约有时会比现货还便宜，此时可以卖出现货，买入合约，等待差价回归。

  2. 可以通过做多永续合约代替现货，可以加3~5倍的杠杆，以实现套利收益的成倍增长，成本是需要支付一些资金费率。也可以直接跨期套利，不需要资金费率成本。

  3. 要每天关注当前的期现差价率，如果达到5%以上，就可以加3倍杠杆去做，除去手续费、滑点、资金费率，搞到10%的收益没问题。

六. 资金费率套利

  1. 统计2019年以来的资金费率套利，BTC和ETH的年化收益均在20%左右，甚至在熊市收益也很少为负。详情参考《资金费率套利研究》。

  2. 虽然长期看总体收益很不错，但实际上收益主要来源于牛市，熊市只能保本，基本没什么收益。比如在2021.12.01~2022.12.31大熊市期间，ETH-U本位的资金费率套利总收益仅为1.63%。除去交易滑点和手续费，勉强保本而已。

七. 合约交易

  1. 2022年1月开始做合约交易，前半年交易很顺利，很快就实现了盈利50%的目标，使用的技术也很简单，主要是亚当理论。然后我开始学习百年一人的文章，研究预测技术。却不想交易成绩开始起起伏伏，折腾几个月，反而没赚到什么钱。

  2. 2022下半年，我接触到段永平的投资理念，才明白我从一开始的稳定盈利到后来的裹足不前的原因：稳定盈利是因为坚持做自己能力圈内的交易，裹足不前是因为我开始突破自己的能力圈，所以看似技术水平进步了，但盈利能力却下降了。

  3. 由此可见，交易要想实现稳定盈利，其实非常简单——只要认清自己的能力边界，只做能力圈内的交易即可。今年我决定将技术分析回归到N字运动和亚当理论上，只做自己最确定的机会。我相信今年一定会有一个更好的成绩！

八. 知识付费

  1. 创造有价值的内容，发布到各大付费阅读平台上。比如Quill: Build your value net on Web3 with Quill

  2. 出售付费资料：只要是不好获取的资料，均可在网上出售。比如由于有些人的微博经常被封，网上就有人整理了李法师Tony的微博，以50元每份的价格出售。我就买了一份。

九. 邀请返佣

  1. 币安的邀请返佣，现货交易永久返佣，合约交易返佣一年。如果我坚持做上三年，一定可以为自己打造一个稳定的现金流收入。

  2. 可以在豆瓣小组内发贴，引用组里的人私信交易，就可以顺势推荐自己的邀请链接了。想做总是有办法的。

基于双币赢的投资策略

发表于 2023-02-03

  1. 双币赢是个非常不错的产品，基于这个产品可以打造一个低风险高收益的投资策略。

  2. 低买ETH：选择您买入目标价，到期时如果ETH未跌到目标价，您将赚取USDT收益；如果ETH跌到目标价，您将成功按目标价买入ETH，并赚取额外ETH。

  3. 高卖ETH：选择您的卖出目标价，到期时如果ETH未涨到目标价，您将赚取ETH收益；如果ETH涨到目标价，您将成功按目标价卖出ETH，并赚取额外的USDT。

  4. 我决定做ETH的双币赢产品，如果低买失败，则可以赚取USDT收益；如果低买成功，则转而做目标价大于等于买入价的高卖产品。如此反复操作。如果无高卖产品可做，就持币待涨，同时可以获得质押收益。价格涨回来只是时间问题。

  5. 上面的策略在牛市和震荡市中，可以获得非常高的收益。但在熊市中，收益会急剧下降，甚至无相应的产品可做。为了解决这一问题，我决定以历史最高价的1/2为分界线划定能力圈，只做目标价在能力圈内的产品，能力圈外的产品统统放弃。这种策略的特点是，当价格在中位线附近震荡时，收益最高；随着价格偏离中位线越远，收益也越低。

  6. 比如目前ETH的历史最高价为$4800，价格中位线为$2400。低买时只做目标价小于$2400的产品，这样即使被套，也只是被套在半山腰，而不会被套在山顶。可以保证一直有合适的低买高卖产品不断地赚取收益。

  7. 另一种避免高位站岗策略是将价格和资金同时分成n等份，第1份资金只做目标价低于(1/n)*H的双币赢产品，第2份资金只做目标价低于(2/n)*H的双币赢产品，第x份资金只做目标价低于(x/n)*H的双币赢产品…这种方法按风险等级分配资金比例，稳扎稳打，步步为营。牛市赚U，熊市赚币，同时会使收益曲线更为平稳。

  8. 比如将资金和价格分成4等份(历史最高价$4800)，第1份只做目标价低于$1200的双币赢产品，第2份只做目标价低于$2400的双币赢产品，第3份只做目标价低于$3600的双币赢产品，第4份只做目标价低于$4800的双币赢产品。

  9. 除了双币赢，还有价差套利、质押收益等赚币方法，可根据市场的实际情况，选择最合适的赚币方法，以实现收益最大化。(完)
