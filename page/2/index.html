双币赢交易实战计划

发表于 2023-03-25

  1. 双币赢是个不错的金融产品，结合质押借币，可以在囤币的同时牛市赚U，熊市赚币。在控制好风险的情况下，拉长时间尺度几乎稳赚不赔，是个打造额外现金流的好工具。

  2. 最近将VEVE的BTC转入了OKX，质押借币了2000U，用于操作BTC双币赢；同时将自己冷钱包的ETH转入了OKX，质押借币了50000U，用于操作ETH双币赢。这是笔不小的资金，需要好好计划。

  3. 用于操作BTC双币赢的2000U，我决定分成4份，每份500U。同时将BTC的价格分成4个区间：0/15000/30000/45000/60000，按价格档位投入相应的资金。
    1）当BTC价格在15000以下时，投入2000U操作双币赢；
    2）当BTC价格在30000以下时，投入1500U操作双币赢；
    3）当BTC价格在45000以下时，投入1000U操作双币赢；
    4）当BTC价格在60000以下时，投入500U操作双币赢。

  4. 用于操作ETH双币赢的50000U，我决定分成5份，每份10000U。同时将ETH的价格分成5个区间：0/1000/2000/3000/4000/5000，按价格档位投入相应的资金。
    1）当ETH价格在1000以下时，投入50000U操作双币赢；
    2）当ETH价格在2000以下时，投入40000U操作双币赢；
    3）当ETH价格在3000以下时，投入30000U操作双币赢；
    4）当ETH价格在4000以下时，投入20000U操作双币赢。
    5）当ETH价格在5000以下时，投入10000U操作双币赢。

  5. 这种阶梯式的投入方式，有助于平衡风险和收益。除了在绝对底部，都始终保持一部分资金闲置，闲置的资金可以用于提前还贷或降低质押率。

我的币圈投资之路

发表于 2023-03-22

  1. 2017年1月，一个偶然的机会，在网上看到OKCoin的广告，开始投资比特币。然后越投越涨，越投越多，最后一发不可收拾，甚至开始网贷买币。到2018年初，已经累积投了40万进去。

  2. 我的想法也很简单，比特币是一个历史性的大机遇。如果成功，那我将收获十倍甚至百倍的收益；如果失败，也就损失一年的工资而已，而且只要我不失业，我的工资完全可以负担起每月的网贷还款。这个风险完全在我的承受范围之内，所以我决定放手一搏！

  3. 当时一心炒币，半夜睡醒还要看一眼行情，加上行情不错，做着暴富美梦的我更无心工作，最终在2018年3月，从龙珠直播离职，失去了每月22K的现金流。当时想着辞职后做NICHE站，照样可以闯出一片天地！

  4. 2018年3月币乎上线，可以通过写文章赚KEY，写了一段时间文章，后来还买了100万KEY点赞赚币。因为竞争激烈，最后并没有赚到什么钱。后来因为玩NeoWorld，又把KEY兑换成了NASH，总体并没有亏钱。

  5. 大约2018年6月份，我开始玩Newworld游戏。好在我时刻保持着风险意识，玩到9月份我就开始陆续退场，将赚到的NASH陆续换成了BTC和ETH。这一战我收获颇多，以ETH币本位来算，币的数量至少翻了一倍。

  6. 2018年一年没有上班，又没什么现金流，再加后下半年跟MMei分手，NASH出金的钱也用来收购MMei的币了，这导致我经济压力越来越大，网贷更是借新还旧。

  7. 这样到了2019年春节后，我终于撑不住了，开始找工作。当时想转型为讲师，于是先去了八维教育，后来又去了积云教育，但由于实在无法适应其企业文化，最后还是应聘到了Bibox做开发，这是一家做数字货币交易所的公司，我很满意。

  8. 由于工作繁忙，2019到2022年，始终没有太多时间研究区块链，这也让我完全错过DeFi爆发的红利，也没有时间撸空投，更不要说研究套利了。事后看，这三年确实错过了很多发财的机会。

  9. 2022年底，公司因FTX事件遭到挤兑倒闭，我也随之失业。坏的一方面是我失去了每月27K的现金流；好的一方面是我终于有时候开始研究DeFi、空投和套利了。

  10. 时间进入到2023年，我开始研究和尝试差价套利，双币赢，ETH循环质押等赚钱方式，希望2023年能收获满满！

基于斗地主的交易策略

发表于 2023-03-05

  1. 最近痴迷于JJ斗地主(连炸模式)，为了能稳定赚金币，尝试了很多方法。功夫不负有心人，目前我已初步达成了这个目标，最近金币数量已经稳定增长到了5w以上。

  2. 我所用的策略非常简单：在坚持“避免大亏”的基础上，同时想办法“提高胜率”。具体方法是及时转出超过上限的金币，同时尽量不叫地主，或最多叫2分地主，以避免大亏。

  3. 只要能做到“避免大亏”，那结果只能是小亏小赚，或是大赚。从最近实践结果来看，平时基本都是小亏小赚，基本持平，偶尔一次大赚，则会成为金币增长的重要来源。

  4. 这样看来，炒币要比斗地主简单得多。一方面炒币可以选择自己最有把握的币种交易，没把握可以不做；另一方面炒币可以很好地控制亏损，严格杜绝大亏的情况出现。

  5. 操作策略上也很好模拟：以日线为例，先选择好币种，然后找一个有利的位置开进去，止损设置为当前K线的最低点，第二天如果未被止损，则将止损上移至新K线的最低点，直到被止损出场。这样遇到好机会就可以大赚一把，而看错时也不会损失太多。

  6. 李法师Tony曾说过“交易的本质就是以小博大，不断地用小额止损去捕捉大行情”，我这个策略完美符合这个理念，而且操作简单，对技术水平也没什么要求。

  7. 段永平的人生哲学是“做对的事情，把事情做对”。而“避免大额亏损”正是交易中对的事情，是交易中的“道”，技术分析只是交易的“术”。只要道对了，赚钱就是早晚的事情。

  8. 为了保证“更健康更长久”的交易，初期应用较小的止损额度(比如1%~5%)，如果能实现稳定盈利，就可以用凯利公式计算出能让资金增长最快的止损额度了。

  9. 实践出真知，我决定先用小资金试验一段时间，如果可以实现稳定盈利，则追加投入。

【宋】欧阳修：卖油翁

陈康肃公善射，当世无双，公亦以此自矜。

科学上网的两种方法

发表于 2023-02-25

一、远程主机

  1. 通过阿里云或腾讯云买一台墙外的Windows主机，然后登录上主机即可直接翻墙。也可以购买一台Ubuntu主机，然后自行安装桌面软件。

  2. 这种方法的优点操作简单，不需要任何配置，也不用担心被封。缺点是使用不方便，低配主机往往分辨率较差。

二、SSH隧道

  1. 购买一台外网服务器：比如可以在阿里云或腾讯云上购买一台新加坡地区的轻量应用服务器，每月仅需30元左右。

  2. 配置SSH免密登录：通过ssh keygen命令生成本地密钥，并将公钥id_rsa.pub中的内容复制到服务器下的.ssh/authorized_keys文件内即可。

  3. 开启SSH隧道：命令ssh -D port username@server_ip，比如ssh -D 2275 lighthouse@129.229.199.197，表示建立一个端口为2275的隧道。

  4. 本地监听上述端口：Mac电脑操作路径为：System Preferences > Network > Advanced > Proxies，勾选SOCKS Proxies并在右侧SOCKS Proxy Server中填入127.0.0.1和2275。然后点击OK返回，再点击Network的Apply即可。

  5. Telegram代理配置：Settings > Data and Storage > Use Proxy > Add Proxy，选择SOCKS5，SERVER输入127.0.0.1，PORT输入2275即可。

  6. 关闭SSH隧道：强制关闭本地命令行窗口即可。

市场走势可以被精确预测吗？

发表于 2023-02-16

  1. 在2023年以前，我一直坚定的认为市场走势是可以被精确预测的。因为2008年我炒股票时，在鼎砥论坛见识了百年一人的全息未来线，对股票市场做出过长期精准的预测，惊为天人！

  2. 截止2022年，百年一人已经研究市场预测24年了，功力已经非常深厚，也从市场上赚到了大量的财富。但看其最近2年的预测，却仍然不尽完美，无法做到100%的确定性。比如2022.02.28预测美元指数已经外强中干了，但事实上那时正是美元指数大涨的起点，一直又涨了7个月才见顶。

  3. 最近沉迷斗地主，我试图在其中找到一种可以稳定获胜的模式，花了很多时间思考和研究，最终也没有成功。因为在一个博弈系统中，参与者的行为会直接影响对手行为，从而再次作用到自身之上，即一但你按预测行事，整个系统就就会发生变化，从而使你的预测失效。

  4. 这让我想到交易市场也是如此，市场走势是多空博弈的结果，而多空博弈又需要保持平衡。一旦大家在某个预期的点位达成一致，那这个位置的多空力量必定失衡，从而导致市场根本无法运行到这个点位。

  5. 由于预测市场的行为会对反过来作用于自身，且这种相互作用的关系会层层嵌套，这导致技术分析永远无法精确地预测市场的走势，只能大致的知晓市场未来的运行状态。理论上来说，只要是由自主个体参与的博弈市场，都是不可精确预测的。

  6. 但这并不能完全否定技术分析的作用，虽然不能精确预测，但模糊预测还是可以的。比如用于判断当前趋势已经运行到了哪个阶段。所以，技术分析仍是多空博弈中非常有效的工具。

如何把子女教育成才？

发表于 2023-02-11

一. 教育大纲

  1. 有条件的话，最少要生2个孩子，一方面筛选比教育更重要，另一方面引入竞争，同时防止单点故障。

  2. 对子女的教育按重要程序依次为：道德教育(道)、品质教育(法)、技能教育(术)。切不可本末倒置！

  3. 教育最重要的是父母要言传身教，以身做则！

二. 道德教育

  1. 道德是人类社会长期进化出的一套获取集体最优解的一种策略，也是获取个人长期最优解的一种策略。因此，一个品行端正的孩子，即使能力平平，长期来看至少生活可以平安顺遂。所以，对孩子的道德教育，要永远要放在第一位。

  2. 道德教育很简单，按社会公认的道德伦理规范教育孩子即可，比如知恩图报、诚实守信、不贪小便宜等。重要的是父母要以身做则，给孩子树立一个好榜样！

三. 品质教育

  1. 品质教育是技能教育的基础，做好了品质教育，则技能教育事半功倍，甚至水到渠成。

  2. 要想做好品质教育也很简单，很多名人传记和成功人物的讲话，都给我们提借了详尽的资料。

四. 技能教育

  1. 天下事有难易乎？为之，则难者亦易矣；不为，则易者亦难矣。人之为学有难易乎？学之，则难者亦易矣；不学，则易者亦难矣。

  2. 技能教育一定要顺势而为，结合孩子的天赋和兴趣爱好。
