打造爆款神标题，只需这一招

发表于 2018-03-27

口号喊得好，生意差不了

小时候，每当放暑假，就会到街上帮父母卖水果(西瓜)。

每当有行人路过，父亲总会喊：“正宗新疆麒麟瓜，不熟不甜不要钱！”

父亲这么一喊，真的有一些人就围过来，看看拍拍、尝尝买买。

我那时候就隐约意识到：口号喊得好，生意就火爆！

做生意时的标语，不就相当于文章的标题吗？

看看下面两幅图，大家体会一下普通标语与神标语的巨大差别。

普通标语

神标语

标题的本质是什么？

怎么样，体会到普通标题与神标题的差别了吗？

要想打造一个爆款标题，就必须要弄明白两个问题：

  1. 标题的本质是什么？
  2. 标题的作用是什么？

我们先看下上面卖西瓜的宣传标语：甜得像杨玉莹，红得像林志铃。

这是什么，这是对用户的承诺：包熟包甜！

标题的本质是对用户的承诺。

如果你是卖西瓜的，你的目的是什么呢？

——当然是吸引顾客来买你的西瓜，这样你才有钱赚嘛！

那文章标题的作用是什么呢？

——当然是吸引用户查看你的文章，看完后关注你，并给你个赞！

标题的作用是吸引用户点击查看文章。

从上面两个角度看，作者与读者之间的关系，是不是与卖西瓜一样，其实是一笔交易？

你给用户标题所承诺的内容，用户拿出自己的时间和精力来阅读，满意的话，就给你点个赞！

作者与读者之间，本质上是一场交易。

你看，很多事情，其实底层都是相通的。

我们不妨拿一篇文章与一个美女作个类比：标题相当于美女的长相，排版相当于美女的穿着，文章的内容嘛(别瞎想)，不妨比作美女的言谈举止。

一招教你起一个神标题

明白了标题的本质与作用，给文章起标题是不是就容易多了？

要给文章起一个合格的标题，就太简单了。

普通标题：你只需要给用户一个承诺。

如果你想起一个爆款标题，也很简单：

爆款标题：你只需要给用户一个无法抗拒的承诺。

请注意：不管你给了用户什么样的承诺，你都一定要做到。如果做不到，你将失去用户对你的信任，以后都不会再有机会。

标题党：标题党只有一次欺骗用户的机会。

你现在再去看看，电视里的广告，电线杆上的宣传单，火车站妈咪的拉客技巧，是不是都充满了这个套路？

郑重声明：本人以前没有，现在不会，以后也不可能去过火车站的小宾馆消费！

总结

当然，要打造一个爆款标题，还有其它很多技巧，但这个是最重要的！

怎么样，有收获吗？

现在是不是已经跃跃欲试了呢？

快去修改你的文章标题吧！

  3. 于是我就拿出7.6万人民币在火币兑换成了USDT，买了42.29个LTC充进去了(2018年1月13号，USDT价格为7.46元/个，莱特币价格为1560元/个)。

三. 放贷总结

成为比特币首富，李笑来用了什么方法？

发表于 2018-03-21

突破局限

最近在学习李笑来老师的课程，在学习过程中，时常为李笑来老师思考问题之深入深刻而惊叹！

他总是能透过纷繁复杂的现象，抓住问题最根本最本质的核心。

我越来越强烈地意识到，李笑来老师能取得今天的成就，能成为比特币首富，是偶然中的必然。

那李笑来老师是怎么遇到比特币的呢？

在一次课程分享中，李笑来老师提到了自己为什么能在2011年就接触到比特币：

在信息获取方面，我有很多我自己的独立思考。

我在Twitter上Follow了1万8千多人，为什么？理由很简单：突破局限。

要不是我这么做，我就不可能在2011年的时候知道比特币这么个东西，也许最终我还是会知道的，比如，2017年。

读完这段话，我特意去查看了李笑来老师的Twitter，李笑来老师关注了1.85万人(如下图)：

李笑来能成为比特币首富，并不像普通人所想象的那样，只是运气好而已。

李笑来的好运气，是他的突破局限的必然结果，而不是原因。

创造好运

在《洛克菲勒的家信》中，有一篇文章叫做《运气靠策划》，洛克菲先生是这么说的：

每个人都是他自己命运的设计者和建筑师。

我不靠天赐的运气活着，但我靠策划运气发达。

等待运气的时候，要知道如何引导运气。

设计运气，就是设计人生。

那么我们该如何创造自己的好运呢？

以下是我的思考出来的3个原则：持续学习、保持开放、保持好奇。

持续学习

我从小就是个特别喜欢学习的人，学习成绩一直名列前茅。从小学到初中，我一直是班里前三名。在高二时进入了年级前10名，学校为了鼓励我们，还带我们去清华北大参观，希望我们能为母校增光添彩。

很可惜，到了高三，因为喝了三鹿奶粉(其实是由于高考制度改革，由文理单科改为大综合)，以做题慢但正确率高为优势的我的成绩便直接下降了一个档次，降到了班级前10名。无法接受这个残酷现实的我，复读了一年，可惜仍未能如愿。

但我好学的品质丝毫没有改变。以至于到了2014年，我干脆给自己取了个网名：好学人(子曰：十室之邑，必有忠信如丘者，不如丘之好学也！)，从杭州搬家到北京时，光书就整整两大箱子(100多本)。

来北京这几年又陆续买了一些，书就更多了。前几天换房子，要把书搬到四楼，搬家师傅是个年轻力壮的汉子，却被一大箱子书压得腿肚子打颤。我们俩合伙搬完了，师傅还是累得足足歇了10多分钟。

持续的学习给我带来了什么好运呢？

我本是个无一技之长的人，也没有什么家庭背景。我从刚毕业时的第一份工作月薪1300块，到今天的22000块。7年时间，我的月薪暴增了17倍。这就是持续学习给我带来好运的最佳证明。

保持开放

我确实是个非常好学的人，但我也是一个极度封闭之人，我觉得开放会给我带来不必要的干扰，影响我的学习时间。

我从高中时代就开始写日记，记录自己的所思所想。这个好习惯一直保持到今天，这期间我研究过许多非常有价值的课题，但我从来都没有公开发表过任何一篇。

正是因为我的这种保守，让我错失了许多机会。比如比特币，一个一有时间就学习的人，竟然不知道已经被电视报道过多次的新事物。

为什么？因为我“两耳不闻窗外事，一心只读圣贤书”，因为我“独学而无友，孤陋而寡闻”。

但上天终究是给了我一个机会。

2014年，我从中国移动辞职，雄心勃勃地准备加入幼儿教育事业。期间遇到一位同事，她告诉我她有个朋友，做IT的，月薪2万多！

我当时保持怀疑，不信行业工资的差距竟然如此之大，但回家后还是上网仔细查阅了资料。

我在知乎上找到了一个案例：一位月薪2万的销售，转行当程序员，仅一年多时间，再次达到了月薪两万！

我信了，并立即采取了行动，变卖家当，进京培训，加入了IT行业。工资也从原来的三五千块一跃过万，因为我较强的学习能力和解决问题能力，月薪又很快从1.5万到2万再到2.2万。

这虽然是一次被动的开放，却也给我人生带来了一次飞跃！

保持好奇

2017年，我第一次接触到比特币，我并没有像大多数人那样，看到就算了，然后继续自己的生活。

我深入研究了比特币，比特币出现的背景，比特币的原理，比特币的愿景！

我被比特币的思想震惊了！

于是我开始了自己的定投之路。之后的事情请参见《进入币圈14个月，我赚了多少钱？》。

虽然我现在并没有赚到多少钱，但我相信，这又是我命运的一个转折点。

后面接触到越来越多的区块链相关的知识，我敏锐地意识到，内容创作者的春天来了！

这不，我已经开始写作了么？

一篇文章只讲一件事

发表于 2018-03-16

一篇文章只讲一件事，但要把这件事讲清楚。

《Unix编程艺术》一书中有句话：Do one thing and do it well.

这句话的意思是一个程序应该只做一件事并把它做好。也就是编程中的单一职责原则。

这个原则对于写作也是通用的，所以写文章时，要坚持一篇文章只讲一件事情，但要把这件事讲得深入彻底。

Markdown

本篇文章由Markdown语法写成

你是否习惯了Markdown简洁强大的语法？

你是否迷恋于Markdown优雅美观的排版？

你是否离不开Markdown高效兼容的格式？

你是否也发现了，币乎的编辑器并不支持Markdown语法！！！

鱼，我所欲也；熊掌，亦我所欲也。

难道二者真的不可兼得吗？ 非也！

Markdown Here

Markdown Here可以将你输入的Markdown文本一键渲染成富文本。

Markdown Here官网地址：https://markdown-here.com/

Markdow Here目前支持Chrome、Firefox、Opera、Safari等浏览器，点击 Get it for Chrome, Firefox, Safari, and Thunderbird 即可跳转到插件下载界面，根据你使用的浏览器下载相应的插件即可。

比如我使用的是谷歌Chrome浏览器，打开后点击 添加至 CHROME 即可。

添加成功后会在浏览器右上角生成一个Markdown Here 小图标。

使用Markdown Here

好了，现在你可以在币乎的编辑器中使用Markdown语法写作了。

点击浏览器右上角的 Markdown Here 小图标 可在编辑模式和预览模式之间切换。

你也可以使用快捷键 Ctrl + Alt + M 快速切换。

好啦，愉快地开始使用Markdown语法写作吧！

如何选出优质项目？

发表于 2018-03-14

一、前情回顾

  1. 2017年的数字货币市场，仿佛新婚的小夫妻，干柴烈火，激情四射！一直在冲刺冲刺再冲刺！高潮一波接着一波！整个币圈都淹没在快感之中。但是，精力再旺盛的小夫妻，也不可能在一直在床上冲刺，对吧？

  2. 2018年，高潮过后，快感消退，市场的热度迅速降温，比特币价格最高时超过13万人民币，前段最低点时，不到4万人民币，差点跌得只剩下个零头。

  3. 很多新入币圈的小韭菜，哪见过种跌法，都慌了神。有的四处求救，寻找心理安慰；有的备受打击，皈依佛系；有放弃治疗，挥刀自宫。

  4. 过去的日子，已成定局，未来的行情，还在不确定之中。生活终究还是要继续。那么2018年，我们应该采取什么样的投资策略呢？

二、大佬观点

  1. 我们先来看看比特币首富李笑来的观点：“首先，建议大多数人不要入局；其次，少投少投再少投。项目数量少投一点，每个项目资金量上少投一点，每个项目比例上少占一点。”李笑来给出这个建议的理由是：2018年以来大量新项目涌入，机构投资者跑步入场。供给增加，而需求却没多大变化，况且大佬们进场不是来接盘的，怎么可能涨呢？

  2. 我们先不做分析和评论，下面再来看看老猫的观点：“未来的一年，是机会之年，也是行业落地之年，大多数的区块链项目会面临价值的拷问，炒作依然会有，但价值将更加凸现，请大家把握好那些『看起来不够刺激却显而易见的机会』，真正的大机会从来都是看起来每个人都有参与机会，最终获得大成果的却只有少数人。”

  3. 最后，我们再来看看量子链创始人帅初的观点：“99%的Token在未来2-3年内，会归零，现在所享受的行业红利和流动性溢价，会消失。行业慢慢进入长跑期，产品比概念重要。不切实际的计划或假大空的项目，会快速挥霍完资金，逐渐失败。”

  4. 三位币圈大佬的观点都很明显，在2018年，依靠炒白皮书、炒投资背景、炒概念圈钱的时代已经过去了。正如新婚的激情期已过，接下来婚姻要进入磨合期，要开始踏踏实实开始过日子，要面对柴米油盐等生活中的实际问题了。

  5. 尽管区块链的革命性和发展前景举世公认，但现在很多币的市值都是虚的，需要很长的时间才能够实现。在2018年，闭着眼买入就能赚钱是肯定行不通了，新项目的ICO也没有必要再去抢了。

三、解决方案

  1. 那我们该怎么办呢？

  2. 其实不论任何投资，要想赚到钱，最重要的就是选择好的投资对象。你想在股市里赚钱，就要选出好公司；你想在楼市里赚钱，就要选出好楼盘；你想在数字货币市场赚钱，就要选出好的项目(Token)。面对鱼龙混杂的数字货币市场，我们应该选择什么样的项目呢？

  3. 提问是解决问题的捷径，选择一个项目前，大家不妨自问以下4个问题。如果这4个问题都能通过，无疑这个项目是值得投资的。

    1、这个项目已经落地了吗？(25分)
    2、这个项目是龙头项目吗？(25分)
    3、这个项目的准入门槛高吗？(25分)
    4、这个项目的成长性如何？(25分)

  4. 根据以上4个问题，我们很容易就能提炼出一套具体的投资策略：我们可以从已落地的项目中，选择出龙头项目，再进一步判断该项目的门槛和成长性。

  5. 下面以比特币举例分析：毫无疑问，在支付领域，比特币是已落地的龙头项目；门槛方面，比特币技术上是没有么门槛的，分叉币一大堆，但比特币作为区块链的世界的第一个应用，有着不可替代的地位；最后看成长性，仅从比特币替代黄金这个方面，比特币仍有巨大的成长性。综合考虑，比特币落地项目25分，龙头项目25分，门槛15分，成长性15分，总得分80分。仍然是非常不错的投资项目。

  6. 当然，在仅在支付领域，又分为大额支付、小额支付、匿名支付等概念，你不妨分别找出其龙头项目分析一下。–END–
