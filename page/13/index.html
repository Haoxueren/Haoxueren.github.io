进入币圈14个月，我赚了多少钱？

发表于 2018-03-20

遇见比特币

2015年，我从杭州来到北京工作，那时候我对编程十分狂热，每天不知疲倦地写代码，上班写代码，下班写代码，周末写代码…

这就样搞了一年多，我把自己感兴趣的东西，想做的项目，都搞了一遍。到了2016年底，公司业务慢慢地稳定了下来，我的工作也变得十分清闲。

我每天做完工作，就东搞搞，西弄弄。于是在一个偶然的机会，我接触到了比特币。

忙，会让你错失很多机会！

认真研究了几天之后，我觉得这个东西很有前途，就跟同事聊了聊这个东西，说我打算投资。

一个同事说：这种东西完全就是炒作，小心亏得裤衩都不剩！

但我相信自己的判断，依然决定投资！

别人的建议可以参考，但决定要自己做！

2017年1月10日，我在OKCoin平台(现在的OKEX)充值了1100块钱，开启了我的第一笔数字货币交易。

定投

当时我的想法很简单，觉得这是个很有前途的新事物，我可以拿出工资的十分之一来定投，即便全部损失了，对我也没什么影响。

好奇才能发现机会，冒险才能利用机会。

于是我就用Python写了个定投脚本，每天定投比特币和莱特币(那时候以太坊还没上火币)。

日子就这样平静地过着，反正脚本每天在定时执行，我并没有花太多时间精力在上面。

之后行情不断地振荡走高，我的生活平淡中带着一些惊喜。

9.4危机

随着比特币一路飙升，相关部门看不下去了，不断地出台限制政策，先是禁止提币，要求实名认证。

后来发现收效甚微，行情反而继续飙升！这是亦裸裸的打脸好么？于是相关部门一声令下，将ICO定义为非法融资，市场应声暴跌！

看一个人的实力，要看他的对手。

但我不为所动，继续我的定投！我依然看好比特币，我甚至加大了定投额度！

相关部门看到我依然如此淡定，心情十分不爽，于是又一纸文书，下令限期关闭境内的数字货币交易所！

交易所开始停止人民币入金，我失去了入场的渠道，这下黔驴技穷了！

于是我把账户中的莱特币，以太坊，全兑换成了比特币，提现到了blockchain.info钱包(当时只支持比特币)。转身投入到了我伟大的IT事业中。

杀不死比特币的，只会使比特币更强大！

ALL IN

就在我快要忘了这个事时，我看到了火币网打出的广告，说人民币又可以买卖币了！我马上去研究了一下，试探性地买了1000元的USDT，发现还不错！虽然有汇率损失，但总算又可以定投了！

我的比特币又回来了！

但这时的我，由于比特币价格的暴涨，对比特币已经有了更深入的理解，更加坚定的信心。我已经不满足于定投了！

我像一匹脱缰的野马，开始疯狂地跑步入场！

从2017年11月6号到2018年1月4号不到两个月时间，我先后投入了18万多！

市场最疯狂的时候，你一定要保持冷静！

期间市场也很配合我，一路飙升，我先后投入的20万资金，最高达到了60万！

我开始幻想着财务自由之后的生活了！

上帝欲使其受伤，必先使其疯狂！

暴跌中的反思

后面的事情大家都知道了，市场开始一路狂跌，惨不忍睹！腰斩已经算好的了，很多币都斩到脚后跟了！

自己对区块链，数字货币前景很看好，这没错！

打算做个长期投资者，做价值投资，这也很好！

唯一做得不好的地方是没有做好资金配置。

我应该坚持定投的，而不是看到行情疯狂就大力ALL IN！

不要害怕错过，机会是等出来的。

今天统计了一下，自己前前后后共投入20.5万，现在还有21.8万，勉强保本(如下图)：

我仍然坚信我当初的判断！

一篇文章只讲一件事，但要把这件事讲清楚。

《Unix编程艺术》一书中有句话：Do one thing and do it well.

这句话的意思是一个程序应该只做一件事并把它做好。也就是编程中的单一职责原则。

这个原则对于写作也是通用的，所以写文章时，要坚持一篇文章只讲一件事情，但要把这件事讲得深入彻底。

你是否习惯了Markdown简洁强大的语法？

你是否迷恋于Markdown优雅美观的排版？

你是否离不开Markdown高效兼容的格式？

你是否也发现了，币乎的编辑器并不支持Markdown语法！！！

鱼，我所欲也；熊掌，亦我所欲也。

难道二者真的不可兼得吗？ 非也！

Markdown Here

Markdown Here可以将你输入的Markdown文本一键渲染成富文本。

Markdown Here官网地址：https://markdown-here.com/

Markdow Here目前支持Chrome、Firefox、Opera、Safari等浏览器，点击 Get it for Chrome, Firefox, Safari, and Thunderbird 即可跳转到插件下载界面，根据你使用的浏览器下载相应的插件即可。

比如我使用的是谷歌Chrome浏览器，打开后点击 添加至 CHROME 即可。

添加成功后会在浏览器右上角生成一个Markdown Here 小图标。

使用Markdown Here

好了，现在你可以在币乎的编辑器中使用Markdown语法写作了。

点击浏览器右上角的 Markdown Here 小图标 可在编辑模式和预览模式之间切换。

你也可以使用快捷键 Ctrl + Alt + M 快速切换。

好啦，愉快地开始使用Markdown语法写作吧！

如何选出优质项目？

发表于 2018-03-14

一、前情回顾

  1. 2017年的数字货币市场，仿佛新婚的小夫妻，干柴烈火，激情四射！一直在冲刺冲刺再冲刺！高潮一波接着一波！整个币圈都淹没在快感之中。但是，精力再旺盛的小夫妻，也不可能在一直在床上冲刺，对吧？

  2. 2018年，高潮过后，快感消退，市场的热度迅速降温，比特币价格最高时超过13万人民币，前段最低点时，不到4万人民币，差点跌得只剩下个零头。

  3. 很多新入币圈的小韭菜，哪见过种跌法，都慌了神。有的四处求救，寻找心理安慰；有的备受打击，皈依佛系；有放弃治疗，挥刀自宫。

  4. 过去的日子，已成定局，未来的行情，还在不确定之中。生活终究还是要继续。那么2018年，我们应该采取什么样的投资策略呢？

二、大佬观点

  1. 我们先来看看比特币首富李笑来的观点：“首先，建议大多数人不要入局；其次，少投少投再少投。项目数量少投一点，每个项目资金量上少投一点，每个项目比例上少占一点。”李笑来给出这个建议的理由是：2018年以来大量新项目涌入，机构投资者跑步入场。供给增加，而需求却没多大变化，况且大佬们进场不是来接盘的，怎么可能涨呢？

  2. 我们先不做分析和评论，下面再来看看老猫的观点：“未来的一年，是机会之年，也是行业落地之年，大多数的区块链项目会面临价值的拷问，炒作依然会有，但价值将更加凸现，请大家把握好那些『看起来不够刺激却显而易见的机会』，真正的大机会从来都是看起来每个人都有参与机会，最终获得大成果的却只有少数人。”

  3. 最后，我们再来看看量子链创始人帅初的观点：“99%的Token在未来2-3年内，会归零，现在所享受的行业红利和流动性溢价，会消失。行业慢慢进入长跑期，产品比概念重要。不切实际的计划或假大空的项目，会快速挥霍完资金，逐渐失败。”

  4. 三位币圈大佬的观点都很明显，在2018年，依靠炒白皮书、炒投资背景、炒概念圈钱的时代已经过去了。正如新婚的激情期已过，接下来婚姻要进入磨合期，要开始踏踏实实开始过日子，要面对柴米油盐等生活中的实际问题了。

  5. 尽管区块链的革命性和发展前景举世公认，但现在很多币的市值都是虚的，需要很长的时间才能够实现。在2018年，闭着眼买入就能赚钱是肯定行不通了，新项目的ICO也没有必要再去抢了。

三、解决方案

  1. 那我们该怎么办呢？

  2. 其实不论任何投资，要想赚到钱，最重要的就是选择好的投资对象。你想在股市里赚钱，就要选出好公司；你想在楼市里赚钱，就要选出好楼盘；你想在数字货币市场赚钱，就要选出好的项目(Token)。面对鱼龙混杂的数字货币市场，我们应该选择什么样的项目呢？

  3. 提问是解决问题的捷径，选择一个项目前，大家不妨自问以下4个问题。如果这4个问题都能通过，无疑这个项目是值得投资的。

    1、这个项目已经落地了吗？(25分)
    2、这个项目是龙头项目吗？(25分)
    3、这个项目的准入门槛高吗？(25分)
    4、这个项目的成长性如何？(25分)

  4. 根据以上4个问题，我们很容易就能提炼出一套具体的投资策略：我们可以从已落地的项目中，选择出龙头项目，再进一步判断该项目的门槛和成长性。

  5. 下面以比特币举例分析：毫无疑问，在支付领域，比特币是已落地的龙头项目；门槛方面，比特币技术上是没有么门槛的，分叉币一大堆，但比特币作为区块链的世界的第一个应用，有着不可替代的地位；最后看成长性，仅从比特币替代黄金这个方面，比特币仍有巨大的成长性。综合考虑，比特币落地项目25分，龙头项目25分，门槛15分，成长性15分，总得分80分。仍然是非常不错的投资项目。

  6. 当然，在仅在支付领域，又分为大额支付、小额支付、匿名支付等概念，你不妨分别找出其龙头项目分析一下。–END–

一枚比特币应该值多少钱？

发表于 2018-03-03

比特币的前世今生

比特币由密码学家Satoshi Nakamoto(中本聪)提出，比特币是一个去中心化的电子货币，总量恒定为2100万枚。

2009年1月3日，比特币创世区块诞生。点击这里查询创世区块信息。

2010年5月22日，一位名叫Laszlo Hanyecz的程序员用10000枚比特币购买了2个披萨。

2017年12月17日，一枚比特币的价格已飙升到近20000美元。

2018年2月6日，短短两个多月的时间，一枚比特币价格又暴跌至6000美元。

你是否迷惑了，比特币到底价值几何？

下面我们就从3个方面来探讨，一枚比特币应该值多少钱。

数字黄金

因比特币数量恒定，不受个别政府国家控制，被人们称为数字黄金。

就目前而言，全球黄金总市值约6~8万亿美元，截止2018年4月20日，比特币的价格为6700美元，总市值为1143亿美元。与黄金市值相比，中间约有60倍的差距。

若以此估算，一枚比特币的价格应该在40万美元左右。

但你我都清楚，比特币比黄金拥有更多的优势。

基础货币

在各大交易所，比特币被作为基础货币已经是司空见惯的事。

比特币被用来衡量其它数字货币的价值，一如外汇市场用美元来衡量其它国家的货币。

2017年，全球美元总量为13万亿，若以此计算，比特币的价格应该在65万美元左右。

收藏品

随着区块链技术的普及，数字货币势必会流行起来，现在全球有60亿人口，而只有2100万枚比特币。

比特币作为数字货币的始祖，一定会具有收藏价值，比特币将奇货可居，拥有比特币会成为身份和地位的象征。

大佬观点

你是否不敢相信未来的比特币将拥有如此大的价值？

让我们一起来看看币圈大佬的看法。

比特币双子星Winklevoss兄弟认为：比特币的市值将最终超过黄金，每一枚比特币的价格将突破30万美元。

拥有20亿Snapchat股权的投资者Jeremy Liew表示：外部因素会驱动人们对比特币的需求，未来10年很可能会将币价推到不可预见的高度。用户数量和虚拟货币的价值密不可分，比特币要想在2030年达到50万美元，用户数量必须占到世界总人口的5%。

再过10年，达到世界总人口的5%，这貌似不是什么难事吧。

综上所述，比特币的实际价值应该在40~60万美元。

由此看来，目前比特币的价格，真的是很便宜很便宜！

当我们把目光放得长远一些，眼前的这些涨涨跌跌就不那么重要了。

因为我们相信时代发展的趋势，我们的目标是星辰大海。

我坚信，在不远的将来，拥有比特币的我们，都将是千万甚至亿万富翁。

以前，我觉得政治最重要，决定了历史的变迁。学了《经济史》，我认识到，经济比政治更重要。政治人物只是舞台上的演员，剧情走向早由经济因素决定了。

2、

举例来说，美国南北战争的根本原因就是经济。北方发展制造业，缺乏雇工，迫切希望南方种植园释放黑奴，让他们到工厂里面当工人。南方庄园主坚决不同意，一旦奴隶制度消失了，谁会愿意忍受烈日高温，到地里采棉花？如果棉花种植成本大幅上涨，就无法向欧洲出口，庄园主就会破产。

再比如，大明王朝的灭亡原因也是经济。历史学家黄仁宇指出，明朝财政非常困难，根本无力抵抗各种入侵。

“在黄仁宇看来，明朝十六世纪的财税困境，在于低下的税收管理水平无法适应其极度复杂的税收结构，导致了财政系统全国范围内的混乱局面。而“奉行成例”的僵化思维又使任何有效的改革无法实行，从而使政府税收长期不足，公共行政能力十分低下，无法有效地行使政府职能，最终导致灾难性的后果。”

3、

很长一段日子，我都持有这种“经济决定论”，进而沉迷于证券市场，相信“股市是经济的晴雨表”。就这样，浪费了很多时间。

后来，全球金融危机爆发，各国纷纷出台刺激经济的政策。这件事让我很困惑：经济学已经有几百年的历史，早就成了一门“科学”，但是为什么没能防止（甚至都没预测到）这么大规模的经济危机呢？

危机爆发之后，经济学家还在争论不休，回答不了几个最基本的问题：

  • 危机会持续多久？
  • 如何解决危机？
  • 类似的危机将来会不会再发生？

我对经济学的信心，就是从那个时候开始动摇和崩溃的。

4、

在学校期间，我学会了制作网页，架设了个人网站。离开学校以后，更是亲身感受到了，互联网技术对整个社会和个人命运的巨大改变。

对我来说，技术变得比经济学更有吸引力了。渐渐地，我不再阅读经济学书籍了，转而大量阅读技术书籍。

熟悉科技史以后，我看待历史的观点，再一次发生了巨大的变化。我现在认为，主导历史的因素，短期（一年到几年）是政治，中期（几年到几十年）是经济，长期（几十年到几百年）则是技术。

长期来看，政治和经济都不太重要，影响不了趋势，真正起决定性作用的是技术。政治和经济只能改变资源的分配和价格，只有技术才能创造出新东西。政治和经济只能使人类在平面上移动，只有技术才能使得人类向上提升，进入下一个阶段。

5、

我发现，人类的每一次大发展，都是技术推动的。

（1）石器时代，人类学会了加工石材，有了最基本的工具，可以定居生活了，进入了氏族社会。

（2）等到掌握了粮食生产技术，人类有了多余的粮食，氏族就分化出了阶层，产生了贵族和首领。

（3）铁器时代，人类掌握了冶炼技术，能生产更好的农具和武器，拥有铁器的氏族开始征服其他氏族，慢慢演变成封建王国。

（4）欧洲的封建时期有1000多年，中国有2000多年，为什么会延续这么长？就是因为这段时间，技术没有重大突破。

（5）等到欧洲人掌握了造纸术和印刷术，新思想第一次可以廉价地大规模传播（此前都是羊皮的手抄书，一般人看不到），黑暗的中世纪因此终结，文艺复兴开始，人类进入近代社会。

（6）火药的出现，使得骑兵和城堡变得毫无作用，一小股军队就会拥有巨大的杀伤力，诸侯林立的欧洲开始统一为一个个民族国家，现代意义上的政府出现了。

（7）工业革命始于人类能够制造蒸汽机，生产力提升了一个数量级，要求大规模的社会分工和交换，由此诞生了资本主义制度，直到今天。

总之，重大的新技术一旦出现，政治制度和经济制度——不管主动或被动——最终都会为之重构，让这项技术的威力发挥到最大。这就是为什么技术是最终的决定因素，决定了人类社会的演变。

6、

今天，我们正身处最新一次、也是最大一次的技术革命：信息技术革命。技术进步的速度日新月异，而且还在加速，这就注定了目前的人类社会也会随之大变。

很多我们习以为常的事情，都已经完全改变了。15年前，世界市值最大的前五大公司只有一家技术公司，今天已经全部是了。新技术创造了财富，财富也在流向技术。

中国也是如此，2017年中国市值最大的公司，是香港上市的腾讯，第二名是美国上市的阿里巴巴。曾经的巨无霸——中国石油、工商银行、中国移动——都被挤到后面去了。

时代已经变了，新人和新公司每天都在崛起，旧的公司和制度每天都在没落。技术正在加速重塑人类社会。

7、

以色列学者尤瓦尔·赫拉利在《未来简史》一书中感叹：

“现代科技已经让人类拥有了超过远古诸神的力量，我们的后代肯定将拥有神一样的创造力和毁灭力。我们并不知道眼前的道路会把我们引向何方，也不知道我们那些像神一般的后代会是什么样子。”

正如他所说，我们正身处在一个前人从未有过的处境：技术让人类这种渺小的生物，拥有了神一般的力量。这种力量会把我们带到哪里？完全不知道。

我的心情很矛盾。一方面，对未来的新技术，我感到无比兴奋和期待，就像推特上面一位网友所说。

“一想到我的寿命只有几十年，我就感到特别难过。倒不是因为我害怕死亡，而是因为我将没有机会看到，未来一千年中各种新奇有趣的新技术。”

另一方面，我还有一种不好的预感。人类掌握的技术，已经强大到足以毁灭人类自身，在技术面前，普通人（大多数人都是普通人）已经没有多大用处了。人类的历史将会怎样改变？这不禁令人充满了在劫难逃的担忧。

阮一峰：熵——宇宙的终极规则

发表于 2017-04-23

1、

有人曾经问我：”成年后，有没有书籍改变过你的世界观？”

我想了想，还真有这样的书。那时，我已经工作好几年了，偶然在图书馆翻到一本旧书《熵：一种新的世界观》(上海译文出版社，1987)。

那本书是科普著作，介绍物理学概念”熵”。中学毕业后，我再没有碰过物理学，但是没想到读完以后，我看待世界的眼光都变了。

“熵”这个概念非常简单，很容易理解，但又异常强大，可以解释很多事情。这篇文章，我就来谈谈，为什么你应该懂得熵是什么，它可能也会改变你的世界观。

2、

为了理解熵，必须讲一点物理学。

19世纪，物理学家开始认识到，世界的动力是能量，并且提出”能量守恒定律”，即能量的总和是不变的。

但是，有一个现象让他们很困惑。

(上图中，单摆在两侧的最高点，势能最大，动能为零；在中间的低点，动能最大，势能为零，能量始终守恒)

物理学家发现，能量无法百分百地转换。比如，蒸汽机使用的是热能，将其转换为推动机器的机械能。这个过程中，总是有一些热能损耗掉，无法完全转变为机械能。

(上图中，能量 E 的转换，总是会导致能量损耗 ∆E)

一开始，物理学家以为是技术水平不高导致的，但后来发现，技术再进步，也无法将能量损耗降到零。他们就将那些在能量转换过程中浪费掉的、无法再利用的能量称为熵。

后来，这个概念被总结成了”热力学第二定律”：能量转换总是会产生熵，如果是封闭系统，所有能量最终都会变成熵。

3、

熵既然是能量，为什么无法利用？它又是怎么产生的？为什么所有能量最后都会变成熵？这些问题我想了很久。

物理学家有很多种解释，有一种我觉得最容易懂：能量转换的时候，大部分能量会转换成预先设定的状态，比如热能变成机械能、电能变成光能。但是，就像细胞突变那样，还有一部分能量会生成新的状态。这部分能量就是熵，由于状态不同，所以很难利用，除非外部注入新的能量，专门处理熵。

(上图，能量转换过程中，创造出许多新状态)

总之，能量转换会创造出新的状态，熵就是进入这些状态的能量。

4、

现在请大家思考：状态多意味着什么？

状态多，就是可能性多，表示比较混乱；状态少，就是可能性少，相对来说就比较有秩序。因此，上面结论的另一种表达是：能量转换会让系统的混乱度增加，熵就是系统的混乱度。

(上图中，熵低则混乱度低，熵高则混乱度高)

转换的能量越大，创造出来的新状态就会越多，因此高能量系统不如低能量系统稳定，因为前者的熵较大。而且，凡是运动的系统都会有能量转换，热力学第二定律就是在说，所有封闭系统最终都会趋向混乱度最大的状态，除非外部注入能量。

(上图中，冰块是分子的有序排列，吸收能量后，变成液体水，分子排列变得无序)

5、

熵让我理解了一件事，如果不施加外力影响，事物永远向着更混乱的状态发展。比如，房间如果没人打扫，只会越来越乱，不可能越来越干净。

(上图中，如果不花费能量打扫，房间总是越来越乱)

为什么”世间好物不坚牢，彩云易散琉璃脆”？就是因为事物维持美好的状态是需要能量的，如果没有能量输入，美好的状态就会结束。

这就是我世界观的变化。我从此认识到，人类社会并非一定会变得更进步、更文明。相反地，人类如同宇宙的其他事物一样，常态和最终命运一定是变得更混乱和无序。过去五千年，人类文明的进步只是因为人类学会利用外部能量（牲畜、火种、水力等等）。越来越多的能量注入，使得人类社会向着文明有序的方向发展。

(上图中，经过130多亿年的膨胀，宇宙变得越来越无序了）

6、

工业革命以后，人类社会的进步速度加快了，变得更加先进有序，消耗的能量也指数级地增长：水力不够了用煤炭，煤炭不够了用石油，石油不够了用核能。

能量消耗越大，就会产生越多的熵。因此，人类社会始终处于一种矛盾状态：整个社会变得更加有序和严密的同时，无序和混乱也在暗处不断滋长。

我们只是依靠更大的能量输入，在压制熵的累积。不断增加的熵，正在各种方面爆发出来：垃圾污染、地球变暖、土地沙化、PM2.5、物种灭绝……甚至心理疾病、孤独感和疏离感的暴增，我认为都是熵的增加对人类精神造成的结果。

我们需要能量，让世界变得有秩序，但这样是有代价的。物理学告诉我们，没有办法消除熵和混乱，我们只是让某些局部变得更有秩序，把混乱转移到另一些领域。

7、

人类社会正在加速发展。表面上，我们正在经历一个减熵过程，一切变得越来越有秩序，自动化带来了便捷。但是，能量消耗也在同步放大，为了解决越来越多的熵，我们不得不寻找更多的能量，这又导致熵的进一步增加，从而陷入恶性循环。

迄今为止，人类一直能够找到足够的能量，解决熵带来的混乱。但是，这种解决方式正变得捉襟见肘。如果我们继续像现在这样加速发展，那么终有一天会出现能量缺口，地球上的能量不足以解决熵，那时一切就会发生逆转，仿佛细小的裂缝演变成巨大的雪崩，秩序开始崩塌，世界走向混乱。
