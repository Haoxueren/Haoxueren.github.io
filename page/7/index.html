如何做出长期正确的决策？

发表于 2021-12-17

如何做出长期正确的决策？

关键点有三个：

1）收集足够多的信息(材料)。
2）进行全面严密的逻辑推理(加工)。
3）从长远的、全局的角度做决策(产品)。

1）收集信息

做决策的第一步就是收集信息了，因为信息是思考决策的原材料。

如果能拥有及时、全面且正确的信息，无疑对作出最佳的决策有非常大的帮助。

那怎样才能收集到足够多的高质量信息呢？

1）要从多个渠道收集信息，比如搜索引擎、论坛、微信群，甚至向专业人士请教等。

2）要从多个方面，多个维度分别收集信息，切免以偏概全。

3）最好将收集到的信息统一整理起来，以备后续使用。

2）加工信息

将第1步收集的信息分类整理好，可以借用笔记软件，以及思维导图，理清事物之间的关系。

然后，凡事多问几个为什么，力求找出变化背后不变的东西，探寻出事物的本质。

也许从不同的角度你会得出不同的结论，没有关系，先将结果记录下来，作为备用选项。

这一步最重要的就是要花足够长的时间去思考，去酝酿。因为只有思考清楚了，做出了正确的决策，后来执行起来才会少走弯路，甚至事半功倍。

面对一些会对我们人生产生深远影响的决策，哪怕花上一年甚至两年的时间去思考，都不算多。

不要着急，一个影响一生的决定，花半年甚至一年时间去思考都是值得的。

3）得出结论

在第1步收集到的信息的基础上，经过第2步反复的思考和推理，就可以得出一系列选择了。而这一步就是要多中选优，做出从长远，从大局方面来，选出的最终的最优解。

这一步最需要注意的就是避免两个大坑：1）短期最优解；2）局部最优解。

怎样才能避开这两个坑呢？

1）站在足够长的时间周期思考问题，比如，站在5年、10年、甚至更长的时间周期去思考问题，才能不被短期利益所诱惑。

2）站在足够高的层次去思考问题，从全局的角度，以发展的眼光去分析，绝不可以让屁股决定脑袋，更不可让情绪代替推理。

你的思考能力才是你的无价之宝，一个正确的决定价值连城。

具体实施上，可以采用四象限图来进行分析，即横轴为短期与长期，纵轴为有利与有弊。这样可以将以上选择分为四类：长期有利、短期有利、短期有弊、长期有弊。

认真分析所有选择的长期与短期利弊，并放置在相应的象限内。

然后选择第一象限的决策就可以了。

我为什么投资以太坊？

1）以太坊首创的智能合约，给区块链行业带来了无限可能，让区块链从功能机时代迈向了智能机时代，为区块链行业应用大爆发奠定了坚定的技术基础。

2）2017年ICO兴起，据统计目前95%的项目都发行在以太坊公链之上，以太坊已经镶嵌到了区块链底层，成为了95%项目的基础设施，这不是哪个打着性能旗号的项目可以轻易取代的。

3）2020年兴起的DEFI，以及2021年兴起的NFT，极大地拓宽了以太坊的应用场景，同时为以太坊带来了第二生长曲线。DEFI与NFT还有很大的发展空间，将持续为以太坊生态贡献力量。

4）以太坊开始获得机构投资者的认可。在机构投资者布局比特币以后，目前越来越多的机构投资者开始布局以太坊，甚至更加看好以太坊的发展潜力。

5）以太坊已经实施了手续费销毁机制(EIP-1559)，并且以太坊正在转POS。这让以太坊很可能成为一项通缩资产，成为比比特币更好的储值手段。另外，以太坊还可以质押或参与DeFi挖矿，对大资金来讲，可以产生相当可观的额外收益(2022.02.17 Staking APR ≈ 5%)，从而实现财务自由。

怎么看待以太坊的增发问题？

比特币总量恒定(2100万)，而以太坊目前每个区块增发一定量的ETH作为手续费，这让以太坊成为了一个总量无上限的币种。2017年我也正是基于这个考虑，并没有大规模投资以太坊。

然而，通胀真的重要吗？他细想想，比特币虽然总量有上限，但目前也通胀啊，目前每个区块增发6.25个BTC，通胀率约为1.74%。所以通胀只是表象，重要的是项目的成长速度。

举例：项目A的成长率为10%，不通胀(通胀率为0%)；项目B的成长率为30%，通胀率为5%。你会选择投资哪个项目呢？

这是显而易见的，项目A的净成长率为10%，项目B的净成长率为25%，当然应该选择投资项目B。

所以，什么增发，总量恒定，甚至总量在不断销毁，都只是现象的游戏而不是核心，问题的核心是项目的成长率与通胀率的差值。

根据ultrasound.money网站的数据，目前以太坊每年需增发540万ETH作为手续费(包含叔块奖励)，目前以太坊总量约1.18亿，所以以太坊的年增发率约为4.57%。

在EIP1559升级后，以太坊每年约销毁330万ETH，则以太坊每年净增发为210万，计算得以太坊的年通胀率为1.8%。这个通胀率已经约等于比特币目前的通胀率(1.74%)。

预计明年以太坊将会进行[The Merge]升级，届时以太坊将由POW全面POS挖矿，预计届时以太坊会由通胀转为负值(-2.4%)，即以太坊会由通胀转为通缩。

也就是说，如果以太坊发展良好，届时以太坊将会成为比比特币更好的储值币种。

多么令人兴奋！

微生物生长曲线

发表于 2021-12-08

任何生物都有出生、发育、繁殖、衰老和死亡的过程。

微生物也不例外，如果把单细胞的微生物接种到体积一定的液体营养液中，在适宜的培养条件下进行培养，这些单细胞微生物就会不断增殖，细胞数目会不断增加，如果把这种增加情况画一个对数曲线，就呈现出一定的规律，叫做单细胞微生物的生长曲线。其纵坐标为每毫升培养液中微生物细胞数目的对数，横坐标为培养时间。生长曲线代表细菌在一个新的适宜的环境中生长繁殖以至衰老死亡的全部过程的动态。

典型的微生物生长曲线包括四个时期：迟缓期、对数期、稳定期、衰亡期。

延迟期：

又叫调整期。细菌接种至培养基后，对新环境有一个短暂适应过程（不适应者可因转种而死亡）。此阶段由于微生物刚刚接种到培养基之上，其代谢系统需要适应新的环境，同时要合成酶、辅酶、其他代谢中间代谢产物等，所以此阶段的细胞数繁殖极少，生长曲线平坦稳定。

对数增长期：

又称指数期。此阶段经过调整期的准备，为此时期的微生物生长提供了足够的物质基础，同时外界环境也是最佳状态。所以此阶段生长速率最快、代谢旺盛、酶系活跃、微生物数量呈指数式增长。

稳定期：

由于培养基中营养物质消耗、毒性产物（有机酸、H2O2等）积累pH下降等不利因素的影响，细菌繁殖速度渐趋下降，相对细菌死亡数开始逐渐增加，此期细菌增殖数与死亡数渐趋平衡。

此阶段活细菌数保持相对稳定、总细菌数达到最高水平、细胞代谢产物积累达到最高峰、是生产的收获期。

衰亡期：

随着稳定期发展，外界环境对继续生长越来越不利、细胞的分解代谢大于合成代谢、继而导致大量细菌死亡。此时细菌进入内源呼吸阶段，细菌死亡速度开始大于新生成的速度、整个群体出现负增长、细胞开始畸形、细胞死亡出现自溶现象，活菌数与培养时间呈反比关系。

租房注意事项

发表于 2021-12-07

1. 不要租老小区的房子

老小区环境、物业不好，而且房子年久失修，隔音效果很差。

我租过一个1986年建成的老房子，结果楼上家里大点声说话楼下都能听到，在楼上走动楼下也能听到脚步声。

所以，2000年以前的老小区，最好都不要租。

2. 不要租刚装修过的房子

刚装修过的房子，往往甲醛严重超标，对身体健康危害极大，说啥也不能租。

3. 房间内网络信号怎么样

有些小区房间内信号极差，上不了网也就算了，甚至接不到电话。

4. 问清楚水电供暖和配套设施

  1. 是否民水民电，怎么缴费，是否有水卡电卡，是否可以网上缴费。

  2. 是否为集体供暖，如果是独立供暖，供暖效果差不说，还需要自己花钱，尽量不要考虑。

  3. 检查热水器、洗衣机、冰箱、燃气等是否可以正常使用。

  4. 是否还有其它隐藏费用。

5. 注意床的质量是否结实平整

人的一天至少有1/3的时间要在床上度过，因此一张好床就显得至关重要。

之前租房就遇到一个铁床的，一翻身就吱吱作响，而且睡上去中间变形，早上醒来腰酸背痛的。

6. 小区周围生活是否便利

  1. 离地铁有多远，1km以内为佳。

  2. 房子在几楼，是否有电梯，没电梯的话楼层三层为佳，低层环境不好，高层及以上每天爬楼梯太累了，而且搬家，取快递都极其不方便。如果有电梯的话，租在中高层为佳。

  3. 小区是否安静，如果靠近马路，或者楼上有小孩子，可能会比较吵，影响作息。

  4. 附近是否有超市，商场，饭店，菜市场，医院等。

7. 注意换房的时间

一般来说，年前的租房淡季，房源多且价格相对便宜；年后为租房旺季，房源少且价格贵。

8. 问清楚转租和退租事宜

计划赶不上变化，也许是对房子不满意，也许是工作变动，也许你新交了男女朋友，都可以会导致你需要换房子。

中途换房子的话，中介费肯定是有去无回了，押金也可能被当作违约金，所以，提前问清楚转租的退租事宜，有备无患。

不会，因为目前房地产已经供大于求了。

1）接盘侠不足。一切红利从本质上讲都是人口红利，而我国的人口出生高峰在1987年左右，往后推20~30年即是房地产行业的高峰(2007~2017)。截止到2021年，这部分人已经34岁了，有能力有意愿买房的，大部分都已经买了。而90后，00后，新生儿数量则呈下降趋势。因此，需求侧不支持房价大幅上涨。

2）供给过剩。前些年国家大力发展房地产，甚至将房地产作为拉动经济发展的手段，导致目前房地产数量供应过剩。这两年我有事去过北京周边的一些地方，几乎全是房地产行业，而其它行业则基本处于调零状态。我们老家的小县城也是如此，到处高楼林立，而新小区的入住率却是相当的低，可以说是十室九空。就拿我2017年买房子的小区来说，一个小区近30栋楼，一梯两户，每栋11层，却只有200多户入住。

2. 那未来房价会大幅下跌吗

短期看不会。

近两年因为全球疫情，各国央行集体大放水，中国也不例外。导致热钱无处可去，今年再次涌入房地产市场，导致一线城市房价再次飙升。

这里需要注意的是，本次房地产行情，上涨的基本都是一线城市，并不是以前的普涨。三四线城市的房子甚至有所下跌。就边北京周边(燕郊)的房子，都下跌了。这说明这轮房地产行情是疫情放水给续的命，是投机行情，而不是真正的买房需求导致的。

3. 那长期来看，中国的房地产会走日本和香港房地产泡沫的老路吗？

会。

中国虽然是有特色的社会主义。但经济规律是不以人的意志为转移的。

不要说中国人多，香港弹丸之地，人口密度比大陆的北上广深更大，房地产泡沫不照样破裂了。日本东京也是如此。

更何况，目前的高房价已经伤害到了国家的根本(人口)，年轻人买不起房，结不起婚，生不起娃。这是执政者绝不可以容忍的。所以，单从人口这一个维度讲，执政者也必须会打击高房价。

房地产泡沫的破裂，将会给社会带来巨大的阵痛，但是，执政者已别无选择。房住不炒，将是未来几十年的主基调。

我们这一代人(80/90)，作为房地产的高位接盘者，将来会有很多人死在房子上。

4. 那我们该怎么办呢？

不买房。买也买最小最便宜的。

在中国，不买房是不实现的。大人可以租房，孩子户口，上学怎么办？没房的话很是麻烦。

所以，如果必须买的话，那不妨买一个最便宜的。给孩子上户口即可。

如何成为一个资本家？

发表于 2021-06-08

1. 什么是资本家？

资本家，顾名思义，就是有资本的人。

具体来说，资本家，只靠自己资产带来的收益，就可以养活自己。

与资本家对应的，就是打工人，打工人不拥有资产或无法靠资产养活自己，必须靠出卖自己的时间精力换取劳动报酬，并以此为生。

简单来说，资本家属于资产阶级；打工人属于无产阶级。

2. 为什么要成为资本家？

为了实现时间自由。

打工人有一个根本性的困境，即他必须出卖自己的时间精力才有收入。无论他打工的收入有多高，一旦他不再出售自己的时间精力，他就会立即失去相应的收入。

这让打工人的生命失去了选择的自由，即所谓生命不止，打工不息。

若是一个打工人喜欢自己的工作，那是非常幸运的；否则，这个打工人一辈子被迫做自己不喜欢的事情，只是为了谋生。

而资本家的处境就截然不同了，资本家的资产带来的收益即可维持自己的生活。从而可以利用自己的时间去做更有价值的事情，自己真正喜欢的事情，并且拥有了时间选择的自由。

3. 如何成为一个资本家？

想办法以最快的速度完成原始积累。

对于白手起家的打工人来说，他进入社会时往往没有任何资本。

这就需要他通过出售自己的时间精力来积累资本。成为打工人或做些小生意都是可以的。

在这个阶段，他应该缩减自己一切不必要的欲望，减少一切不必要的开支，以提高完成原始积累的速度。

甚至，有必要的话，他应该冒一定的风险，以博取超额的利润。

4. 资本积累阶段应该注意哪些事情？

不要掉进消费主义和享乐主义的陷阱。

打工人完成原始积累的两个最大障碍，就是消费主义和享乐主义。

打工人的每一分钱，都是用自己的宝贵的时间(生命)换来的，这些钱就像春天的种子，如果你因为推崇及时行乐这些钱消费掉了，那你就相当于吃掉了春天的种子，等到秋天到来时，你将一无所获。

这也正是很多人中年危机的根源。

资本家每年资产的增值，就远远超出打工作人一年的收入，甚至超出很多倍。

资本家的这些钱，甚至可以说是大风刮来的，人家用用这些钱去买一些奢侈品，完全可以理解。

但打工人如果为了追求所谓的生活品质，就东施效颦，无疑是杀鸡取卵，是非常愚蠢的。

5. 比完成原始积累更重要的是什么？

运用资本的能力。

金钱是最聪明的事物，你必须有足够的智慧，它才会留在你身边。

所以，在拥有一定的资本时，必须同时拥有管理这些资本的能力。否则，这个社会有一万种方法重新让你变得一文不铭。

所以，在积累资本的同时，更重要的是提升自己的认知，积累自己的智本。
