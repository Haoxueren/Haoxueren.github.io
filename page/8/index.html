牛市中的资金轮动规律

发表于 2019-05-19

  1. 在资本市场中，聪明的资金总是先知先觉，而傻钱总是无脑跟风。所以，牛市资金轮动的过程，其实就是资金从优质币向垃圾币流动的过程。

  2. 在牛市初期，进来的资金都是一些聪明钱，这些钱自然会选择买入一些被低估的优质币(一般是龙头股)。所以，在牛市初期，总是优质币先涨。反之，牛市初期就涨的币，基本可以断定是优质币。

  3. 等聪明钱布局完毕后，基本就到了牛市中期，这时最好的投资标的已被聪明钱瓜分完毕，再投资这些标的的性价比已经大大降低。这时闻风而动的资金只能去投资一些潜力股，比如龙头股的竞争者。

  4. 到了牛市末期，进来的资金都是看见赚钱效应而无脑跟风进来的钱，到这个时期，龙头股和潜力股已经涨得差不多了，没有太多上升空间了。这时无脑跟风的资金只好去炒一些仍处于低位的垃圾币和空气币。

  5. 等到垃圾币和空气币开始暴涨的时候，说明资金已经无处可去，因为市场已经没有合适的投资标的了，这时一般已经到了牛市盛宴的末期，接下来就是结账买单的时刻了。

  6. 下面计算了本轮行情中主流数字货币的回血百分比，算法如下：
    回血指标 = (最新价 - 前低点) / (前高点 - 前低点)

    基于以上分析，再结合币种回血幅度统计表，可以得出结论：
    优质币：BTC、ETH、EOS、LTC、BNB
    垃圾币：QTUM、NEO、OMG、IOTA

  • 注：等到QTUM、NEO、OMG、IOTA这4个垃圾币都开始暴涨的时候，就说明牛市已经接近尾声了。

如何写出一份好简历？

发表于 2019-05-16

简历的本质是什么？

简历的本质是求职者与面试官的沟通媒介。

因此，一份好的简历一定是简明扼要，重点突出的。这样才可以把面试官最需要的信息以最快的速度呈现到他面前。

一份简历的最终使命是获得面试官的认可，而展示自己的技术实力只是获取面试官认可的手段。

一定不要本末倒置了。

面试官的需求是什么？

求职面试过程中，主要有两个面试官，即人力面试官和技术面试官。

人力面试官的作用是对简历进行硬性筛选，而技术面试官则对面试能否通过起着决定性作用。

那么，技术面试官的需求又是什么呢？

技术面试官的需求是寻找一位熟悉项目中所用技术，上来就能干活的同事。同时也要保证日后能与自己和谐相处。

在满足以上条件的基础上，才会考虑求职者技术牛不牛，学习能力强不强等因素。

如何写出一份好简历？

基本以上分析，如何写出一份好的简历呢？

主要分为以下3步：

1）简单介绍下个人基本信息。

2）重点罗列自己的专业技能。

3）介绍自己的个人作品&项目经验。

然后就可以了。

为什么说牛市已经来了

发表于 2019-02-25

说牛市已经来了，其实不够准确。因为上一轮牛市太猛烈太持久了，一些赚了大钱的人买别墅买豪车买游艇，导致大量的资金消费到了币圈外。

市场失血过多，一时半会儿缓不过来。类似的情况还有2007年A股的大牛市，如今10年过去了，还没缓过劲儿来。

准确地说，是一波大行情来了，一波月线级别的行情。

原因有以下3点：

1. 基本面

经历了2018年暗无天日的暴跌，比特币跌了80%以上，其它币基本都跌了90%以上。从历史经验来看，泡沫已被挤干净了，已经跌无可跌了。

特别2018年11月份的主流币暴跌，也是市场见底的重要标志。

因为主流币的共识最强，也被大多数人所看好，不到迫不得已，是不会将手里的主流币出售的。所以，如果主流币开始暴跌，基本可以确实，这是空方最后的宣泄了。

砸完主流币以后，基本就跌无可跌了。因为空方手里，没筹码了。

2. 技术面

技术分析这个东西，有人奉之为圭臬，有人弃之如敝履。

其实无论是基本面分析还是技术面分析，都是认识事物的重要维度。二者相辅相成地使用，才是最好的选择。

当然了，每个人的技术系统不同，得出的结论也千差万别。但总体而言，K线也遵循物理学的宏观可测，微观不可测的规律。

即越是大周期，越可测；越是小周期，越是不可测。比如，做技术分析的话，月线就比日线靠谱得多，如果再到分时线，基本就相当于布朗运动了。

我对BTC、ETH的月线进行了一波技术分析，发现了非常明确的见底信号。

3. 消息面

看一则2019-01-29日的币圈快讯：

2018年大小额财富地址数形成背离
据Chaindigg报告显示，2018年链上财富地址数分布出现分歧，小额财富地址从年初开始稳步增加，但在10月之后出现反转下降，而大额财富地址从年初开始表现平稳，10月中旬后开始急剧攀升。大小额财富地址数形成明显背离。
数据很好的反映出，在2018年10月份市场出现了明显的风格转换，短期市场信心下挫带动小额地址离场，而长期资金在“最寒冷”的时候积极入场布局，看好中长期行情。

虽然说消息都是有目的的，甚至是为大资金服务的。但消息也反应了市场的情绪，你如果每天坚持阅读币圈资讯的话，应该能感受得到，市场情绪确实在好转。

比如ETH升级，bakkt交易所，以及比特币ETF等。虽然都是一波三折，但总体是向好的。

总结一下，我认为市场将会开启一波行情，原因有以下3点：

  1. 币价已经下跌到位
  2. 技术形态支持上涨
  3. 大资金布局，市场情绪回暖

嗯，一点个人分析，不构成投资建议。

如有不同观点，欢迎大家在留言区与我探讨！
