香农半仓投资法

发表于 2022-06-26

一、策略简介

  1. 网格交易法的思路来源于信息论之父——香农。

    在任何一个价位，用资金的50%买入资产作为起始仓位，当价格上涨一定幅度就卖出一部分仓位套现，当价格下跌一定幅度就买入一部分仓位补仓。保持仓位和现金的比例始终为50%:50%。香农就是采用这种半仓投资的方式，在资本市场获得了年化29%的好成绩。

  2. 让我们举个例子来具体看看。假如我准备投资 1万 美元囤 ETH，当前 ETH 的价格是 1000 美元。我花 5000 美元买入 5 枚ETH，剩下5000美元现金备用。此时，我账户里的现金价值：ETH 价值 = 50% : 50%。

  3. 如果以太坊涨到 2000 美元一枚，我账户里的现金价值仍为 5000 美元，而 ETH 价值则为 10000 美元。此时我应该卖掉价值 2500 美元的 ETH(1.25枚)。操作完后，我持有 3.75 ETH，市值为 7500 美元，现金价值 7500 美元，二者仍然为 1:1。

  4. 反之，如果以太坊跌到 500 美元，5 ETH 的市值仅为 2500 美元，而我持有 5000 美元的现金。遵循半仓网格的原则，我就需要花 1250 的现金去购买相应数量的 ETH，让我的现金价值：持币价值 = 50% : 50%。此时，我持有 7.5 枚ETH，和 3750 美元的现金。

二、数据分析

  1. 如下表，为在比特币月线级别使用香农囤币法的收益回测(每月收盘时调整一次仓位)与普通囤币的收益(数据来源:huobi.pro)。

  2. 由图表可以看出，使用香农囤币法可以有效地控制回撤，在2019.01月，普通囤币的收益为-74.07%，而香农囤币法的收益为-44.69%；在2020.03月，普通囤币的收益为-51.29%，而香农囤币法的收益为-18.11%。至少可以将最大回撤降低一半以上。

  3. 但是，在大牛市行情，香农囤币法也会极大地降低最高收益，比如在2021.11月，普通囤币的收益为+356.32%，而香农囤币法的收益仅为+185.83%，约为普通囤币法的1/2。可以说香农囤币法几乎是起到了一个削峰填谷的作用。

  4. 经统计，在54个月中，普通囤币的月均收益为53.8%，而香农囤币法的月均收益仅为39.5%。说明普通囤币的整体收益要好于香农囤币法。但其中有36个月香农收益超过了基准收益，即在66.66%的时间内，香农囤币法的收益都要好过普通囤币。

  5. 可以看出，在震荡市场，香农囤币法可以反复高抛低吸，扩大收益；在单边下跌行情中，香农囤币法会阶梯加仓，摊低成本；但在单边上涨行情中，香农囤币法会不断减仓，降低收益。建议追求稳健收益的用户选择香农囤币法。不在乎收益波动的采用普通囤币法，遇到单边上涨行情，可以获得更大的收益。

  6. 有一个更好的方法是，先判断当前处于牛市还是熊市。如果处于牛市，则使用普通囤币法。如果处于熊市，则使用香农囤币法。即使判断错了，只是浪费些时间或少些收益而已，并不会造成太大的损失。

1）交易最关键的是风险控制！

2）要时刻牢记，你可能是错的！

N字运动交易系统

发表于 2022-03-08

  1. 只做第三段行情，不见兔子不撒鹰！

  2. 百年一人说，市场最基本的运行方式为N之运动。经过我大量观察，同一级别运动最明显的特征是运行时间会大体相近，利用这一点可以判断市场处于N字运动的哪个阶段(N1-N2-N3)。

  3. 只做第三段行情，是因为可以通过第1段行情和第2段行情的具体走势，来判断第3段行情的起始点和大体运行空间。在波浪理论中，就是只做波浪理论推动浪中的三浪或调整浪中的C浪(如图)。

  1. 这种交易手法最大的优点，在于可以过滤掉所有的逆势交易和弱势交易，始终保证自己是在顺势而为。

  2. 当市场开启第1段行情时，要按兵不动，观察第一段的运动的速度，并耐心等待后续的调整，然后观察市场第2段运动的速度，结合第1段行情和第2段行情形成的结构(上涨结构/下跌结构)，再决定是否参与第3段行情。

  3. 在上涨趋势中，最好的形态为市场快速上涨后强势调整(比如横盘)，一旦调整结束，市场必定会再次上涨；同理，在下跌趋势中，市场在暴跌后弱势反弹，一旦反弹结束，市场必定会再次下跌。

  4. 尊重市场，彻底放弃抄底摸顶的想法，耐心等到行情启动后再入场，可以大大提高交易的胜率。比如在见到多头的第一次激励后再入场做多，或见到空头的第一次警告后再入场做空。

  5. 有所舍，必有所得！

合约交易备忘录

发表于 2022-03-06

品种筛选

目前支持U本位合约的品种非常多，在交易时要优先选择强势的品种做多，弱势的品种做空。同时选择走势比较规则的品种，以方便进行技术分析。

技术分析

在选定要交易的标的后，要对交易标的多个周期(15M,1H,4H,1D)进行全面深入的技术分析，通过大周期判断市场趋势，通过小周期寻找市场买卖点。

比如通过4H周期判断趋势，通过1H周期寻找买卖点。

1）做空单时，要确保4H周期正处于下跌周期，且1H周期的上涨周期已耗尽，正在转为下跌周期，止损点可设为4H波段高点。

2）做多单时，要确保4H周期正处于上涨周期，且1H周期的下跌周期已耗尽，正在转为上涨周期，止损点可设为4H波段低点。

市场涨跌的本质是多空双方力量的博弈，在多头或空头展示了自己的实力后，再入场交易，往往是最安全的入场点。

比如在看到大阳线后再入场做多，因为大阳线往往预示着多头对空头有着压倒性的优势。

同理，在看到大阴线后再入场做空，因为大阴线往往预示着空头对多头有着压倒性优势。

交易日记

每次下单前都要写下交易理由，比如：

1）4H周期：当前处于下跌周期，预计还将下跌m个周期。
2）1H周期：目前上涨周期已耗尽，预计将下跌n个周期。

每次交易结束后，都要及时写交易日记(交易理由、交易记录、交易总结)，并定期回顾，以达到不断积累交易经验，提高交易水平的目的。

最后，不要频繁交易，等待(时间成本)也是交易的成本之一。

财务自由之后做什么？

中国未来人口发展趋势

发表于 2022-01-18

先看看最近几年的新生儿出生数量数据：

1987年：2508万
2017年：1723万
2020年：1200万
2021年：985万

2021年相比35年前(1987年)下降了60%，相比5年前(2017年)下降了43%，相比1年前(2020年)下降了18%

这个生育率下降速度，让国家领导都坐不住了！

当前婚育现状观察

从1978年改革开放后，中国已经高速发展了40多年，到目前为止，增量红利基本已消耗殆尽，社会开始进入存量竞争阶段。在增量发展阶段，你好我好大家好。但到了存量发展阶段，由于资源有限，往往是你多我少，甚至是你死我活的竞争。

具体表现有以下几种：

1）婚育率不断下降：目前90后的年龄区间为22~32岁，而据统计，目前90后的结婚率却只有10%，更可怕的是离婚率暴涨，目前中国近年的离结比已经突破了50%，传统的婚姻关系在新的社会形态下难以维系。

2）躺平族的出现：由于存量竞争的残酷性，导致部分看不到希望的年轻人开始躺平，不婚不育不消费的思想开始蔓延。

3）女权主义盛行，性别对立严重。

这其中最值得警惕的就是不婚不育思想的蔓延，是关系到国家和民族未来的大事，那么，照此发展下去会怎么样？

我们不妨做一个思想实验：

如果所有人都不婚不育会怎么样？结论很简单，没有新生人口，国家和民族走向消亡。如果全人类都不婚不育会怎么样？结论也很简单，人类走向消亡。所以，从这个角度来讲，提倡不婚不育者就是反社会分子，甚至是反人类分子。

但所有人都不婚不育，显然是不可能的。结局就是一部分人选择结婚生育，另一部分人选择不婚不育。那么50~60年后，这一代人消失，选择结婚生育的人留下有婚育思想的后代，而选择不婚不育的人就此消失，这些人的思想连同基因片段被无情地从人类基因库中删除。

优胜劣汰，适才生存。自然规律像一把无形的手，它不会跟你讲道理。

所以，我们该怎么办？

作为中华好儿女，我们当然要战胜万难，结婚生子。

也希望国家给力点，控制房价，提升老百姓的生活质量，让老百姓结得起婚，养得起娃。

人口下降的本质原因

随着科技的发展，生育率下降并非中国独有，在全球是普遍现象。

这里不得不说下人口下降的本质原因。

真正导致人类生育率不断降低的罪魁祸首，是科技的发展挤占了人类的生存空间。

科技的发展在给人类生活带来更多便利的同时，机器在不断将人类从越来越多的岗位剔除出去，越来越多的事情掌握在机器手中。人类的生存空间在不断缩小。这跟我们把森林砍了以后，野生动物的数量会下降是一个道理。

生产效率越高，这个世界对人口的需求量就越小，这也是任何事情都逃不过供需平衡法则。

也许有一天，人类会从这个世界消失，社会的发展也从生物文明进入机器文明。

人类是进化的手段，而不是最终的目的。

不过那是我们后代需要考虑的事情了。
