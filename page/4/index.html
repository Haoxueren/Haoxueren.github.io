我的技术分析系统

发表于 2022-08-19

我的技术分析系统主要由三部分组成：位置、结构、强弱。

一）位置：高位、低位

1）高位：当市场经过长期上涨或大幅度上涨，多头力量往往会进入老阳阶段，此时要注意阴极阴生。

2）低位：当市场经过长期下跌或大幅度下跌，空头力量往往会进入老阴阶段，此时要注意阴极阳生。

二）结构：N字运动

1）市场运动的最基本结构是N字运动，一个完整的N字运动由N1→N2→N3组成。有时两个N字运动会连接在一起，组成一个五波运动，即N1→N2→N3→N4→N5。波浪理论可以看作是对N字运动的简化分类(并不全面，不包括多个N字运动相连的情况)。

2）N字运动的一个重要性质是：N字运动的各部分之间具有可公度性。通过这种可公度性，可以计算出N字运动可能的趋势转换点。

3）亚当理论揭示了N字运动三部分之间最重要的关系，即N1与N3关于N2呈中心对称，N1与N3在运动时间和空间上均具有相似性。

三）强弱：多空博弈

1）市场涨跌背后的的原因是多空博弈，多空博弈的过程形成了市场的波动结构。

2）涨跌的持续时间，可以理解为多空双方的兵力；涨跌的力度代表多空双方的火力。根据多空博弈的兵力和火力，可以判断多空力量的强弱。

3）在兵力充足但火力不足的情况下，往往表现为趋势犹豫或波折(趋势推进困难)；在火力充足但兵力不足的情况下，往往表现为脉冲式的上涨或下跌(持续时间不够)，往往会被很快打回原型。

4）通过相邻两个波段涨跌时间和涨跌力度的比较，可以很好地判断出当前是多头主场还是空头主场。通过最近两个同向波段的涨跌时间和力度，可以判断出当前是推动结构还是衰竭结构。

暴涨暴跌行情分类研究！

发表于 2022-08-18

少阳阶段暴涨，回调可入场做多；少阴阶段暴跌，反弹可入场做空。

1）暴涨指短时间内上涨空间巨大，也就是上涨速度极快。暴涨背后的原因是多头能量对空头能量有着压倒性优势。

2）暴跌指短时间内下跌空间巨大，也就是下跌速度极快。暴跌背后的原因是空头能量对多头能量有着压倒性优势。

3）暴涨暴跌行情按所处位置可分为高位和低位。市场处于高位，往往意味着多头已经进入上涨末期(老阳阶段)；市场处于低位，空头往往已经进入下跌末期(老阴阶段)。以此维度进行完备分类，有以下4种情况：

3.1）高位暴涨：行情很可能已经进入N3甚至是N5末期(冲顶阶段)，此时做多的不确定性已经大增。若随后横盘调整，后续可能会再次上冲；若随后持续下跌且时间和空间上均超过之前任何一次下跌，则大概率意味着趋势已经逆转。总之，高位暴涨，不宜追高！

3.2）低位暴涨：往往是阴极阳生，若上涨时间和幅度均超过之前任何一次反弹，则很可能意味着下跌趋势已经逆转，根据回调力度可以考虑入场做多；若上涨时间持续较短(涨幅巨大反而使空间一步到位)，后续也很可能重归跌势，这种情况不可追多！

3.3）高位暴跌：高位暴跌，往往是空头的第一次警告，说明空头的力量已经开始崭露头角(阳极阴生)，后续即使大幅回升，在前高附近也可能会被空头再次暴击！若随后弱势反弹，更说明多头很弱，连一次有规模的反攻都组织不起来了，往往代表多多头大势已去！

3.4）低位暴跌：当市场经过长时间下跌或大幅下跌，空头力量往往已经处于老阴阶段，此时的空头力量已是强弩之末，外强中干！所以低位暴跌，很可能是空头的最后宣泄！后续若是强力反弹，则说明物极必反，阴极阳生！若后续弱势反弹，也不可追空，因为空间上已经跌无可跌！这时往往是多头仍需时间积累力量，等待时机反攻！

4）总结：少阳阶段暴涨，可入场做多；少阴阶段暴跌，可入场做空；其余两种情况，放弃交易！

4.1）若市场高位暴跌且本次下跌的时间和力度均超过之前每一次下跌，则很可能意味着上涨趋势已逆转，若随后的反弹力度较弱，则可以考虑入场做空。

4.2）若市场低位暴涨且本次上涨的时间和空间均超过之前每一次反弹，则很可能意味着下跌趋势已逆转，根据之后的回调力度，可以考虑入场做多。

4.3）其余两种情况，放弃交易！放弃并不会导致任何亏损！强行交易却会！

把一招练一万遍！

发表于 2022-08-18

模式化交易，把一招练一万遍，是实现稳定盈利的可靠方法！

1）从我2022.01.17开始做合约以来，按交易的盈利状况，大体可分为两个阶段。

2）阶段一(2022.01.17~2022.04.28)期间，我的交易非常顺利，短短3个多月时间，我的累积盈利就达到了50%(每次交易的风险额度为5%)，按盈亏比计算胜率在73%左右。

3）阶段二(2022.04.28~2022.08.12)期间，交易可以说是相当不顺利，时赚时亏，累积盈利一直没能有效突破，甚至还一度回撤到了10%附近，总体盈亏比胜率也下降到了50%左右。

4）为了找出交易盈亏比下降的原因，我进行了长期深入的反思，我认为导致第二阶段盈利上不去的根本原因是——随着我对预测技术的研究，交易模式也在不知不常中发生了根本性的转变——由模式化交易转变为了预测式交易，也就是根据预测抄底摸顶。

5）在交易的第一阶段，我基本只做暴涨暴跌形态(暴涨后强势调整或暴跌后弱势反弹的形态)，这种情况下多空力量差异悬殊，趋势的方向也很明显，此时入场交易胜算很高。而在交易的第二阶段，我是以时间周期分析为基础进行交易的，主要采用主导周期、顶底循环、黄金分割等技术手段寻找趋势的转折点，完全抛弃了阶段一的交易模式。

7）所以，要想稳定盈利，有一个非常简单可靠的方法——把一招练一万遍(模式化交易) 。比如只做暴涨后的第三段行情，和暴跌后的第三段行情，把这两招练好了，就可以轻松实现稳定盈利(一招鲜，吃遍天)。

8）这(模式化交易)可以说是通往稳定盈利的平坦大道了！不要舍近求远！所以接下来我决定放弃周期预测方面的研究，专注于暴涨暴跌后的走势研究。等我财务自由(资金突破1000万)后再考虑研究预测技术。

追踪止损交易系统

发表于 2022-08-15

  1. Tony说交易的秘诀是以小博大，我认为很有道理。交易如果能截断亏损，放任利润的话，长期来看肯定是能盈利的。

  2. 交易就是在不断波动的行情里找一个有利的位置开进去，错了就砍仓，对了就设置一个幅度不断加码，以开仓价作为平仓价。虽然十次有九次都会被平仓出局，大多数时间都无任何收获，但只要一年下来做对两波大行情就足够你十年不用工作。

  3. 那么开仓后错误的标准是什么呢？我认为可以将跌破上根K线的最低点作为多单的错误标准，将突破上根K线的最高点作为空单的错误标准。那么加仓的标准呢？可以在第1单脱离成本线后，加仓第2单(同样将止损设置在上根K线的极值点)，以些类推。

  4. 这种交易策略可以用最小的风险跟随住趋势，很好地实现看错时少亏钱，看对时多赚钱的目标，遇到极端行情还能大赚一笔。操作上也很简单，只要在每根K线结束后，将止损移动到上根K线的最低点或最高点即可。

  5. 理论上来讲，这种策略非常适用于单边行情，在震荡行情中可能会表现不佳。根据我以往的交易经验，震荡行情多发生在顶部或底部，因此可以等到顶部或底部被确认后再入场，胜算会更高。当然，如果能通过技术分析避开震荡行情，那是最好的。

  6. 根据我自身的实际情况，每次交易可以使用1%~5%的止损额度。等交易一段时间，就可以根据实际情况再作调整。我认为这个交易策略非常值得一试，可以分配一定的风险额度与等额止盈止损策略同步并行，实践出真知！

用最小的风险博取最大的收益

发表于 2022-08-12

  1. 如何实现稳定盈利？这是我一直苦思冥想的问题。也许我应该放弃追求“稳定盈利”的想法，毕竟，胜可知而不可为。只要能盈利就好，管它稳定不稳定呢？

  2. 李法师Tony说：“交易就是在不断波动的行情里找一个有利的位置开进去，错了就砍仓，对了就设置一个幅度不断加码，以开仓价作为平仓价。虽然十次有九次都会被平仓出局，大多数时间都无任何收获，但只要一年下来做对两波大行情就足够你十年不用工作。”

  3. 这条微博可以说某种程度上道出了交易的真谛——用最小的风险博取最大的收益。甚至用零风险博取收益。同时也是一种立于不败之地再求胜的策略，做对了有收益，做错了也几乎没什么损失。非常好！

  4. 实际交易中由于交易手续费的存在，开仓时无法自带平价止损，但可以通过调整仓位将止损额度限定在最低水平(用蚂蚁仓试水)，一旦单子有一定幅度的盈利，即可将止损移动到保本线，并同步开启浮盈加仓机制。

  5. 如果后续行情向不利的方向发展，则最多触发平价止损保本出场。而如果行情向有利的方向发展，则可以按一定幅度浮盈加仓，并同步移动止损线。直到达到预期的趋势结束时间平仓出场。

  6. 结合我自身的实际情况，我决定将每次交易的最大风险额度控制在1%以内(通过调整仓位保证必要的止损空间)，一旦盈利达到初始风险额度的10%，则自动将止损位调整到保本线(可通过”追踪止损-只减仓”实现)，若行情继续向预期方向发展，则盈利每增加10%，自动浮盈加仓一次。

  7. 这种方法有效限制了每次交易的最大亏损额度，很好地实现了“截断亏损，让利润快跑”的交易理念。

马丁格尔网格交易策略

发表于 2022-06-27 更新于 2022-08-11

一. 基本原理

  1. 投资标的价格在上涨和下跌的空间上具有不对称性。比如一个币种现价为10，往下最多下跌到0，而往上却可以涨到20、30，甚至50，上涨空间远大于下跌空间。如果以1:1的盈亏比做多该标的，则止盈的概率将远大于止损的概率。当止损价格逼近0时，交易的胜率也将逼迫100%。

  2. 利用投资标的下跌空间有限，上涨空间无限这一不对称性，结合马丁格尔仓位管理，可以打造出一套稳赚不赔的交易系统。这套系统对投资标的只有两个要求：1.标的价格必须有波动；2.标的价格不能归零。除此之外，无需依赖其它的任何假设。

  3. 找出一个或多个长期来看价格不会归零的投资标的，可以说是很简单的一件事，而标的的价格在不断波动，这是个显而易见的事实。所以，这种策略的生命力可以说是非常顽强了！

二. 注意事项

  1. 这种交易策略的关键是要选择一个成长性好，波动率大，且长期看不会归零的品种。在币圈，最靠谱的选择就是BTC和ETH，这两个币种经受住了市场的长期考验，已经嵌套进了区块链的底层系统，长期来看是最不可能归零的币种，成长性也有一定的保证。

  2. 当然，为了避免单点故障，可以挑选2个或更多的品种实践这种策略，安全系数将会大大提升。

三. 交易计划

  1. 以当前价格x为基准，将价格平均分成n档(每档间距为1/n)，止盈止损分别在本档价格上加减1/n，对应的盈利期望依次翻倍，即[1, 2, 4...]。根据每档的开仓价(止盈止损)和盈利期望可以计算出本档应开的仓位。

  2. 按生成的交易计划依次开单，若第1档交易亏损，则启动第2档交易，若第2档亏损，则启动第3档交易…依此类推，直到进入最后一档交易(最后一档交易止损为0，必定会以盈利收场)。若中间任一档交易盈利出场，则本轮交易结束。

  3. 理论上每一档交易均应设置止盈止损，在止损后下一档重新开仓(应开数量)。但实际上无须设置止损，在达到止损价后，在上一档遗留仓位的基础上直接加仓即可(实开数量)！

五. 总结

  1. 以上就是这种稳赚不赔的交易系统介绍和使用方法，其本质是利用涨跌空间的不对称性(有下限无上限)，用时间换空间，达到出手必赢的效果。若配合技术分析使用，效果更佳。这不正是我梦寐以求的确定性吗！

  2. 市场总是在不停在牛熊中转换，在熊市低部区域，可以大胆地入场交易，就能在毫无风险的情况下赚到30%甚至100%，可以说非常棒了！

  3. 而且这种策略可用于币本位交易，用于增加币本位的数量。比如可以通过交易ETH/BTC增加BTC的数量，也可以通过交易UNI/ETH增加ETH币本位的数量。(完)

香农半仓投资法

发表于 2022-06-26

一、策略简介

  1. 网格交易法的思路来源于信息论之父——香农。

    在任何一个价位，用资金的50%买入资产作为起始仓位，当价格上涨一定幅度就卖出一部分仓位套现，当价格下跌一定幅度就买入一部分仓位补仓。保持仓位和现金的比例始终为50%:50%。香农就是采用这种半仓投资的方式，在资本市场获得了年化29%的好成绩。

  2. 让我们举个例子来具体看看。假如我准备投资 1万 美元囤 ETH，当前 ETH 的价格是 1000 美元。我花 5000 美元买入 5 枚ETH，剩下5000美元现金备用。此时，我账户里的现金价值：ETH 价值 = 50% : 50%。

  3. 如果以太坊涨到 2000 美元一枚，我账户里的现金价值仍为 5000 美元，而 ETH 价值则为 10000 美元。此时我应该卖掉价值 2500 美元的 ETH(1.25枚)。操作完后，我持有 3.75 ETH，市值为 7500 美元，现金价值 7500 美元，二者仍然为 1:1。

  4. 反之，如果以太坊跌到 500 美元，5 ETH 的市值仅为 2500 美元，而我持有 5000 美元的现金。遵循半仓网格的原则，我就需要花 1250 的现金去购买相应数量的 ETH，让我的现金价值：持币价值 = 50% : 50%。此时，我持有 7.5 枚ETH，和 3750 美元的现金。

二、数据分析

  1. 如下表，为在比特币月线级别使用香农囤币法的收益回测(每月收盘时调整一次仓位)与普通囤币的收益(数据来源:huobi.pro)。

  2. 由图表可以看出，使用香农囤币法可以有效地控制回撤，在2019.01月，普通囤币的收益为-74.07%，而香农囤币法的收益为-44.69%；在2020.03月，普通囤币的收益为-51.29%，而香农囤币法的收益为-18.11%。至少可以将最大回撤降低一半以上。

  3. 但是，在大牛市行情，香农囤币法也会极大地降低最高收益，比如在2021.11月，普通囤币的收益为+356.32%，而香农囤币法的收益仅为+185.83%，约为普通囤币法的1/2。可以说香农囤币法几乎是起到了一个削峰填谷的作用。

  4. 经统计，在54个月中，普通囤币的月均收益为53.8%，而香农囤币法的月均收益仅为39.5%。说明普通囤币的整体收益要好于香农囤币法。但其中有36个月香农收益超过了基准收益，即在66.66%的时间内，香农囤币法的收益都要好过普通囤币。

  5. 可以看出，在震荡市场，香农囤币法可以反复高抛低吸，扩大收益；在单边下跌行情中，香农囤币法会阶梯加仓，摊低成本；但在单边上涨行情中，香农囤币法会不断减仓，降低收益。建议追求稳健收益的用户选择香农囤币法。不在乎收益波动的采用普通囤币法，遇到单边上涨行情，可以获得更大的收益。

  6. 有一个更好的方法是，先判断当前处于牛市还是熊市。如果处于牛市，则使用普通囤币法。如果处于熊市，则使用香农囤币法。即使判断错了，只是浪费些时间或少些收益而已，并不会造成太大的损失。
