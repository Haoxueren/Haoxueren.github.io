牛市中的大小年现象

发表于 2023-10-30

  1. 小时候家里种苹果，我对苹果树的大小年现象印象深刻。后来我发现，股市和币市中也同样存在大小年现象。以下为苹果树大小年的介绍：

    苹果树的大小年现象，即一年结果多，一年结果少，是果树生长发育过程中常见的现象。这种现象主要是由于苹果树的生理特性和环境因素造成的。在一年中，苹果树花量大、结果多，开花结果消耗大量的营养，造成其他器官特别是枝条顶芽得到养分大量减少，致使当年因营养不足无法形成花芽，所以大年之后为小年。相反，小年花果少，树体营养物质积累增多，为花芽分化创造了良好的条件，所以下年产量大增，即为大年。

  2. 2017年为比特币的大牛市，在新高的基础上涨了近20倍；2021年为比特币的小牛市，在新高的基础上仅涨了3位多；可以推断，2025年又该是比特币的大牛市了，突破新高后再涨10倍还是非常有希望的。即比特币会涨到60万美元左右，以太坊会涨到5万美元左右。

  3. 若比特币能涨到60w美元左右，那比特币的体量，完全可以作为国家的外汇储备，以及国家之间的结算货币了，这很可能是2025年牛市的主要叙事，到2025年牛市末期，甚至会出现比特币冲击百万美元的呼声。我预计，取代美元的下一个国际货币，不会是人民币，而是比特币！

  4. 在A股市场，2007年为大牛市，8年后的2015年为小牛市，如今2023年正好又8年过去了，我预计A股也将迎来一波像2007年一样的大牛市！A股万点不是梦！

下轮牛市的起点在哪里？

  1. 以史为鉴，可以知兴替！历史与未来的关系，就像同一套算法的不同结果。是参数的变化导致了结果的变化，但其背后的算法是不变的。所以认真研究历史，就可以通晓未来！

  3. 第一轮牛熊周期共持续47个月(2013.12~2017.12)：其中下跌期13个月(2013.12~2015.01)，震荡期7个月(2015.02~2015.08)，上涨期27个月(2015.09~2017.12)。

  4. 第二轮牛熊周期共持续46个月(2018.01~2021.11)：其中下跌期13个月(2018.01~2019.01)，震荡期14个月(2019.02~2020.03)，上涨期19个月(2020.04~2021.11)。

  5. 目前处于第三轮牛熊周期中，已经走完了下跌期14个月(2021.11~2022.12)，目前处于震荡期的第5个月，预计再震荡3~9个月就会启动新一轮牛市，对应牛市启动的时间为2023.08~2024.02，根据总周期测算，本轮牛熊周期将至少运行46~47个月，下轮牛市见顶的时间预计为2025.09~2025.10。

  6. 结论：下轮牛市的起点区间在2023.08~2024.02，下轮牛市的终点区间在2025.09~2025.10。

不确定性中必有确定性！

发表于 2023-03-29

  1. 任何确定的事情中，都必有不确定的部分；任何不确定的事情中，都必有确定性的部分。二者就像太极中的阴阳仪，相互依存，不可分割。

  2. 投资市场的走势，看似有很强的不确定性，但其中必有可以确定的部分。而交易就是在不确定性中寻找确定性，抓住确定性，并靠确定性赚钱。以下是几个例子。

  3. 任意一个币，未来能涨多高是不确定的，但能跌多低是确定的(最多归零)。我就是根据这个原理，设计了《必胜网格交易系统》，只要币价还会上下波动，就能盈利出局。

  4. 币圈的涨跌节奏是不确定的，但牛市之后必有熊市，熊市之后必有牛市。只要囤住币耐心等待，就一定能等到牛市的高光时刻。这也是囤币党的理论基础。

  5. 下轮牛市什么时候来，精确的时间是不确定的，但时间区间是确定的，就在2023年1~12月。下轮牛市能涨多高是不确定的，但比特币和以太坊突破前高几乎是确定的。

  6. 期现套利的原理也是如此，币价的涨跌是不确定的，但随着交割时间的临近，期现的差价的回归是确定的。

  7. 还有些事情，短期看是不确定的，长期看是确定的。比如以太坊在2023年底前，不一定能涨到3000U以上，但在2025年底前，则几乎一定能涨到3000U以上。
