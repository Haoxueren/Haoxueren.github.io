中国历史：夏朝简史

发表于 2023-05-07
  • 注：本文由 Perplexity.ai 辅助收集资料。

一、夏朝简介

  1. 夏朝（约公元前2070年-约公元前1600年）是中国历史记载的第一个中原部族世袭制朝代，​共传14代、​17后，​建立者为大禹。​夏朝的政治形态是由多个部落联盟或复杂酋邦形式联合组成，​其主体部族称为夏人。

  2. 夏朝位于今天的中国中部地区，​大致以洛阳为中心的伊洛河流域。​其疆域北部到达山西省南部，​西部到达黄河、​潼关一带，​东部到开封一带，​南部大概到达伏牛山一带。

  3. ​夏朝初期，​夏部落与非夏部落犬牙交错的杂居景象，​族群之间的斗争连绵不断。​夏部落最大的威胁来自东方的夷部落，​展开了百余年激烈的夷夏之争。

  4. ​夏朝的历史记载最早出现于西周，​《逸周书·度邑》、​《论语》、​《竹书纪年》和《史记·夏本纪》等历史文献都有关于夏朝的纪年与世系的记载。​

二、重要事件

  1. 大禹治水：大禹是夏朝的建立者，​他成功治理了上古大洪水，​为夏朝的繁荣奠定了基础。​

  2. 太康失国：夏朝启的儿子太康继位后，​沉湎于声色酒食，​导致国家衰败。​

  3. 后羿代夏：夏朝末期，​夏启昏庸无度，​后羿作为东夷部落的首领，​通过驱逐太康夺得夏朝的权位。​

  4. 少康中兴：少康是夏朝的一位君主，​他成功振兴了夏朝，​使国家重新繁荣。​

  5. 商汤灭夏：公元前1600年左右，​因夏桀昏庸无道，​商汤带领商部落灭掉夏朝，​建立了商朝。

  3. 第一轮牛熊周期共持续48个月(2013.12~2017.12)：其中下跌期14个月(2013.12~2015.01)，震荡期8个月(2015.02~2015.08)，上涨期28个月(2015.09~2017.12)。

  4. 第二轮牛熊周期共持续46个月(2018.01~2021.11)：其中下跌期13个月(2018.01~2019.01)，震荡期14个月(2019.02~2020.03)，上涨期20个月(2020.04~2021.11)。

  5. 目前处于第三轮牛熊周期中，已经走完了下跌期14个月(2021.11~2022.12)，目前处于震荡期的第5个月，预计再震荡3~9个月就会启动新一轮牛市，对应牛市启动的时间为2023.08~2024.02，根据总周期测算，本轮牛熊周期将运行46~48个月，下轮牛市见顶的时间预计为2025.09~2025.11。

  6. 结论：下轮牛市的起点区间在2023.08~2024.02，下轮牛市的终点区间在2025.09~2025.11。

不确定性中必有确定性！

发表于 2023-03-29

  1. 任何确定的事情中，都必有不确定的部分；任何不确定的事情中，都必有确定性的部分。二者就像太极中的阴阳仪，相互依存，不可分割。

  2. 投资市场的走势，看似有很强的不确定性，但其中必有可以确定的部分。而交易就是在不确定性中寻找确定性，抓住确定性，并靠确定性赚钱。以下是几个例子。

  3. 任意一个币，未来能涨多高是不确定的，但能跌多低是确定的(最多归零)。我就是根据这个原理，设计了《必胜网格交易系统》，只要币价还会上下波动，就能盈利出局。

  4. 币圈的涨跌节奏是不确定的，但牛市之后必有熊市，熊市之后必有牛市。只要囤住币耐心等待，就一定能等到牛市的高光时刻。这也是囤币党的理论基础。

  5. 下轮牛市什么时候来，精确的时间是不确定的，但时间区间是确定的，就在2023年1~12月。下轮牛市能涨多高是不确定的，但比特币和以太坊突破前高几乎是确定的。

  6. 期现套利的原理也是如此，币价的涨跌是不确定的，但随着交割时间的临近，期现的差价的回归是确定的。

  7. 还有些事情，短期看是不确定的，长期看是确定的。比如以太坊在2023年底前，不一定能涨到3000U以上，但在2025年底前，则几乎一定能涨到3000U以上。

双币赢交易实战计划

发表于 2023-03-25

  1. 双币赢是个不错的金融产品，结合质押借币，可以在囤币的同时牛市赚U，熊市赚币。在控制好风险的情况下，拉长时间尺度几乎稳赚不赔，是个打造额外现金流的好工具。

  2. 最近将VEVE的BTC转入了OKX，质押借币了2000U，用于操作BTC双币赢；同时将自己冷钱包的ETH转入了OKX，质押借币了50000U，用于操作ETH双币赢。这是笔不小的资金，需要好好计划。

  3. 用于操作BTC双币赢的2000U，我决定分成4份，每份500U。同时将BTC的价格分成4个区间：0/15000/30000/45000/60000，按价格档位投入相应的资金。
    1）当BTC价格在15000以下时，投入2000U操作双币赢；
    2）当BTC价格在30000以下时，投入1500U操作双币赢；
    3）当BTC价格在45000以下时，投入1000U操作双币赢；
    4）当BTC价格在60000以下时，投入500U操作双币赢。

  4. 用于操作ETH双币赢的50000U，我决定分成5份，每份10000U。同时将ETH的价格分成5个区间：0/1000/2000/3000/4000/5000，按价格档位投入相应的资金。
    1）当ETH价格在1000以下时，投入50000U操作双币赢；
    2）当ETH价格在2000以下时，投入40000U操作双币赢；
    3）当ETH价格在3000以下时，投入30000U操作双币赢；
    4）当ETH价格在4000以下时，投入20000U操作双币赢。
    5）当ETH价格在5000以下时，投入10000U操作双币赢。

  5. 这种阶梯式的投入方式，有助于平衡风险和收益。除了在绝对底部，都始终保持一部分资金闲置，闲置的资金可以用于提前还贷或降低质押率。

基于斗地主的交易策略

发表于 2023-03-05

  1. 最近痴迷于JJ斗地主(连炸模式)，为了能稳定赚金币，尝试了很多方法。功夫不负有心人，目前我已初步达成了这个目标，最近金币数量已经稳定增长到了5w以上。

  2. 我所用的策略非常简单：在坚持“避免大亏”的基础上，同时想办法“提高胜率”。具体方法是及时转出超过上限的金币，同时尽量不叫地主，或最多叫2分地主，以避免大亏。

  3. 只要能做到“避免大亏”，那结果只能是小亏小赚，或是大赚。从最近实践结果来看，平时基本都是小亏小赚，基本持平，偶尔一次大赚，则会成为金币增长的重要来源。

  4. 这样看来，炒币要比斗地主简单得多。一方面炒币可以选择自己最有把握的币种交易，没把握可以不做；另一方面炒币可以很好地控制亏损，严格杜绝大亏的情况出现。

  5. 操作策略上也很好模拟：以日线为例，先选择好币种，然后找一个有利的位置开进去，止损设置为当前K线的最低点，第二天如果未被止损，则将止损上移至新K线的最低点，直到被止损出场。这样遇到好机会就可以大赚一把，而看错时也不会损失太多。

  6. 李法师Tony曾说过“交易的本质就是以小博大，不断地用小额止损去捕捉大行情”，我这个策略完美符合这个理念，而且操作简单，对技术水平也没什么要求。

  7. 段永平的人生哲学是“做对的事情，把事情做对”。而“避免大额亏损”正是交易中对的事情，是交易中的“道”，技术分析只是交易的“术”。只要道对了，赚钱就是早晚的事情。

  8. 为了保证“更健康更长久”的交易，初期应用较小的止损额度(比如1%~5%)，如果能实现稳定盈利，就可以用凯利公式计算出能让资金增长最快的止损额度了。

  9. 实践出真知，我决定先用小资金试验一段时间，如果可以实现稳定盈利，则追加投入。

科学上网的两种方法

发表于 2023-02-25

一、远程主机

  1. 通过阿里云或腾讯云买一台墙外的Windows主机，然后登录上主机即可直接翻墙。也可以购买一台Ubuntu主机，然后自行安装桌面软件。

  2. 这种方法的优点操作简单，不需要任何配置，也不用担心被封。缺点是使用不方便，低配主机往往分辨率较差。

二、SSH隧道

  1. 购买一台外网服务器：比如可以在阿里云或腾讯云上购买一台新加坡地区的轻量应用服务器，每月仅需30元左右。

  2. 配置SSH免密登录：通过ssh keygen命令生成本地密钥，并将公钥id_rsa.pub中的内容复制到服务器下的.ssh/authorized_keys文件内即可。

  3. 开启SSH隧道：命令ssh -D port username@server_ip，比如ssh -D 2275 lighthouse@129.229.199.197，表示建立一个端口为2275的隧道。

  4. 本地监听上述端口：Mac电脑操作路径为：System Preferences > Network > Advanced > Proxies，勾选SOCKS Proxies并在右侧SOCKS Proxy Server中填入127.0.0.1和2275。然后点击OK返回，再点击Network的Apply即可。

  5. Telegram代理配置：Settings > Data and Storage > Use Proxy > Add Proxy，选择SOCKS5，SERVER输入127.0.0.1，PORT输入2275即可。

  6. 关闭SSH隧道：强制关闭本地命令行窗口即可。
