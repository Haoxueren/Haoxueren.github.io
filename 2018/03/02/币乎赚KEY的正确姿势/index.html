币乎赚KEY的正确姿势

发表于 2018-03-02

一. 什么是KEY？

KEY是币乎基于以太坊发行的ERC20代币，总量恒定为1000亿。

KEY的用途还是很多的，比如可以用来给你喜欢的文章点赞，付费提问，加入收费圈子等。

当然，最重要的，把你的KEY锁定起来，还可以赚利息！

二. 如何获取KEY？

  1. 注册后实名奖励：新用户注册后，完成实名认证奖励送7万KEY，目前KEY在OTCBTC平台上的价格为0.05元人民币(最新价格)，7万KEY相当于3500元人民币！心动不如行动，马上去注册吧！千万别告诉我你不差这点钱哦！

  2. 邀请朋友注册奖励：目前邀请朋友注册，前20名有奖励，累计最高奖励7万KEY。马上发消息给你的兄弟姐妹，同学同事，七大姑八大姨吧！不要等机会错过了，才坐在那里扼腕叹息！

  3. 发表文章或评论奖励：如果你不会写文章，评论总会写吧？只要你写的东西被别人点赞了，你就有钱可赚！对，就是 so easy！

  4. 给别人点赞的奖励：你没看错，你给别人点赞也有奖励！这真是懒人的福音啊！当然，点赞也是有技巧的，大体的原则是你点赞后点赞的人越多，你的收益就越高。

  5. 锁定KEY的奖励：如果你是个日理万机的大忙人呢？放心，币乎也没有放弃你。你可以通过在账户中锁定KEY获取收益。奖励池释放的KEY，会拿出10%按比例分配给锁定KEY的人。是不是觉得很美妙？

最后，想要获取更准确，更详细的信息，请点击币乎帮助中心查看。
