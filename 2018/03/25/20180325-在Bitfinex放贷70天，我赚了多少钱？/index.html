在Bitfinex放贷70天，我赚了多少钱？

发表于 2018-03-25

一. 惊人的利率

  1. 2017年10月份，Bitfinex平台的放贷的日利率一度高达0.21%，折合年利率76%，绝逼的高利贷啊，再牛逼的P2P也得甘拜下风吧！

  2. 面对这么赚钱的好事儿，我怎么会无动于衷呢？于是，2018年年初，我也放了1万美元在Bitfinex平台放贷，想想每天都能有几美元的睡后收入，做梦都美滋滋！

  3. 但近期随着币价的一泻千里，放贷利率也跟着自由落体。今天我登录进账户看了下，放贷利率(日利率)只有可怜0.017%了(折合年利率6%左右)。这点利率，聊胜于无了，适逢近期币乎开放充提KEY，我准备取出来去屯KEY了！

  4. 截止今天，放贷2个多月，做个总结吧！

一. Bitfinex放贷

  1. 2017年12月份，我从网上了解到，通过Bitfinex平台，可以放贷赚利息。于是就兴冲冲地跑去注册账户，但当时数字货币市场太火爆了，Bitfinex竟然宣布新用户太多了，服务器压力太大，暂停注册！

  2. 一直到2018年1月13号，我发现Bitfinex又开放注册了，但账户至少要充值1万美元的资产才能被激活。这时Bitfinex平台放贷的日利率大约是0.07%左右，我掐指一算，年利率25%左右(单利)，比余额宝4.1%的年利率，强了一二三四五六倍啊！

  3. 于是我就拿出7.6万人民币在火币兑换成了USDT，买了42.29个LTC充进去了(2018年1月13号，USDT价格为7.46元/个，莱特币价格为1560元/个)。

三. 放贷总结

  1. 截止到2018年3月24日，整整70天。那这70天内，我一共赚了多少钱呢？

  2. 在Bitfinex放贷70天，共赚121.87美元(严格来说，并没有赚到钱，因为入场时USDT溢价较高，约7.46元一个，现在USDT只有6.47元了)。

  3. 但是，谁还会意这些小事呢？因为，我马上就是坐拥百万KEY大户啦！
