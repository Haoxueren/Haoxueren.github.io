如何选出优质项目？

发表于 2018-03-14

一、前情回顾

  1. 2017年的数字货币市场，仿佛新婚的小夫妻，干柴烈火，激情四射！一直在冲刺冲刺再冲刺！高潮一波接着一波！整个币圈都淹没在快感之中。但是，精力再旺盛的小夫妻，也不可能在一直在床上冲刺，对吧？

  2. 2018年，高潮过后，快感消退，市场的热度迅速降温，比特币价格最高时超过13万人民币，前段最低点时，不到4万人民币，差点跌得只剩下个零头。

  3. 很多新入币圈的小韭菜，哪见过种跌法，都慌了神。有的四处求救，寻找心理安慰；有的备受打击，皈依佛系；有放弃治疗，挥刀自宫。

  4. 过去的日子，已成定局，未来的行情，还在不确定之中。生活终究还是要继续。那么2018年，我们应该采取什么样的投资策略呢？

二、大佬观点

  1. 我们先来看看比特币首富李笑来的观点：“首先，建议大多数人不要入局；其次，少投少投再少投。项目数量少投一点，每个项目资金量上少投一点，每个项目比例上少占一点。”李笑来给出这个建议的理由是：2018年以来大量新项目涌入，机构投资者跑步入场。供给增加，而需求却没多大变化，况且大佬们进场不是来接盘的，怎么可能涨呢？

  2. 我们先不做分析和评论，下面再来看看老猫的观点：“未来的一年，是机会之年，也是行业落地之年，大多数的区块链项目会面临价值的拷问，炒作依然会有，但价值将更加凸现，请大家把握好那些『看起来不够刺激却显而易见的机会』，真正的大机会从来都是看起来每个人都有参与机会，最终获得大成果的却只有少数人。”

  3. 最后，我们再来看看量子链创始人帅初的观点：“99%的Token在未来2-3年内，会归零，现在所享受的行业红利和流动性溢价，会消失。行业慢慢进入长跑期，产品比概念重要。不切实际的计划或假大空的项目，会快速挥霍完资金，逐渐失败。”

  4. 三位币圈大佬的观点都很明显，在2018年，依靠炒白皮书、炒投资背景、炒概念圈钱的时代已经过去了。正如新婚的激情期已过，接下来婚姻要进入磨合期，要开始踏踏实实开始过日子，要面对柴米油盐等生活中的实际问题了。

  5. 尽管区块链的革命性和发展前景举世公认，但现在很多币的市值都是虚的，需要很长的时间才能够实现。在2018年，闭着眼买入就能赚钱是肯定行不通了，新项目的ICO也没有必要再去抢了。

三、解决方案

  1. 那我们该怎么办呢？

  2. 其实不论任何投资，要想赚到钱，最重要的就是选择好的投资对象。你想在股市里赚钱，就要选出好公司；你想在楼市里赚钱，就要选出好楼盘；你想在数字货币市场赚钱，就要选出好的项目(Token)。面对鱼龙混杂的数字货币市场，我们应该选择什么样的项目呢？

  3. 提问是解决问题的捷径，选择一个项目前，大家不妨自问以下4个问题。如果这4个问题都能通过，无疑这个项目是值得投资的。

    1、这个项目已经落地了吗？(25分)
    2、这个项目是龙头项目吗？(25分)
    3、这个项目的准入门槛高吗？(25分)
    4、这个项目的成长性如何？(25分)

  4. 根据以上4个问题，我们很容易就能提炼出一套具体的投资策略：我们可以从已落地的项目中，选择出龙头项目，再进一步判断该项目的门槛和成长性。

  5. 下面以比特币举例分析：毫无疑问，在支付领域，比特币是已落地的龙头项目；门槛方面，比特币技术上是没有么门槛的，分叉币一大堆，但比特币作为区块链的世界的第一个应用，有着不可替代的地位；最后看成长性，仅从比特币替代黄金这个方面，比特币仍有巨大的成长性。综合考虑，比特币落地项目25分，龙头项目25分，门槛15分，成长性15分，总得分80分。仍然是非常不错的投资项目。

  6. 当然，在仅在支付领域，又分为大额支付、小额支付、匿名支付等概念，你不妨分别找出其龙头项目分析一下。–END–
